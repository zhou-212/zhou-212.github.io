亚马逊欧洲站 8 大合规名词一次搞懂:GPSR / CE / EPR / VAT 都啥意思

刚开欧洲站的小白一定被 GPSR / CE / EPR / VAT / EPREL / IOSS / PPWR / EUDR 这串缩写搞晕。这张卡用大白话讲清楚 8 个最常见名词,各是谁关心、什么时候管,3 分钟读完心里有底。

一句话结论

欧盟合规看着复杂,其实核心就 8 个名词:GPSR(产品安全)、CE(合格标志)、EPR(生产者责任)、VAT(增值税)、EPREL(能效数据库)、IOSS(进口一站式申报)、PPWR(新包装法)、EUDR(去森林砍伐)。搞清楚这 8 个,80% 的新手疑问都能解决。

这是啥

你第一次打开亚马逊欧洲站后台,大概率会看到一堆”EU Compliance”、“EPR Registration”、“Responsible Person”弹窗,邮件里时不时蹦出 PPWR、CRA、IOSS 这种缩写。这不是吓你,是欧盟法规的常态:他们喜欢用缩写,全称一大串,就连德国本地小卖家也得谷歌半天。

这张卡就是帮你”破冰” —— 8 个最常见的名词,每个用 3 句话说清楚:这是啥 / 谁关心 / 什么时候管。读完你应该能看懂亚马逊邮件、听懂合规顾问报价、对上号。

8 个名词 + 1 张表

1. GPSR(General Product Safety Regulation,通用产品安全法规)

  • 这是啥:2024-12-13 全面生效的欧盟产品安全”总章程”,要求所有非食品类消费品在欧盟销售时,必须有一个”欧盟责任人(Responsible Person)” —— 这个人 / 公司要在欧盟有真实地址,能替消费者和市场监管机构”兜底”。
  • 谁关心:所有在 Amazon EU 卖非食品消费品的卖家(电子、玩具、家居、美妆、户外、宠物…);亚马逊后台专门有”EU Responsible Person”字段要填。
  • 什么时候管:现在就在管。没填的 ASIN 已经被限流甚至下架,2024 年底就发生过大面积 listing 消失。详见 GPSR 法规卡。

2. CE(Conformité Européenne,欧合格证)

  • 这是啥:商品上的合格证,证明这件产品符合欧盟的健康 / 安全 / 环保要求,可以合法在欧洲卖。强制性,几乎所有非食品电子 / 玩具 / 机械 / 医疗器械 / 个人防护装备都必须有。
  • 谁关心:产品制造商 / 品牌方(贴 CE 标志的”责任人”);亚马逊对缺 CE 的产品也会下架。
  • 什么时候管:上架前必须有。CE 标志要打 5mm 高以上,详细怎么打见 CE 标识怎么打。注意”China Export”陷阱:长得像 CE 但其实不是,详见那篇卡。

3. EPR(Extended Producer Responsibility,生产者延伸责任)

  • 这是啥:卖东西的人要为产品的”全生命周期”埋单 —— 包括包装回收、电子废弃物处理、纺织品循环、电池回收等。各国分别有 EPR 注册制度,包装类必注册(德国 LUCID、法国 Citeo、意大利 CONAI …)。
  • 谁关心:所有在欧盟销售带包装商品 / 电气电子 / 纺织品 / 电池的卖家。亚马逊后台”Manage your EPR compliance”是必填项。
  • 什么时候管:每个站点开张前就要有。没注册号,亚马逊会屏蔽你的 listing,直到你补上。

4. VAT(Value Added Tax,增值税)

  • 这是啥:欧盟成员国对商品 / 服务征收的”消费税”,类似中国增值税。跨境卖家在欧盟有库存(比如用 FBA)或销售额超当地远程销售起征点(目前 €10,000/年,2025 年改革后将调整),必须注册 VAT 号并按期申报。
  • 谁关心:所有在欧盟有库存或超阈值销售的卖家;每个国家 VAT 号不同(德国 DE、法国 FR、意大利 IT、西班牙 ES …)。
  • 什么时候管:FBA 入仓前就要有。德国 VAT 必须先注册,否则仓库不接收入库。

5. EPREL(European Product Registry for Energy Labelling,欧盟能效产品数据库)

  • 这是啥:欧盟的”能效标签数据库”,卖家要先把产品(空调、冰箱、灯泡、电视、烘干机…)信息登记到 EPREL,系统生成一张能效标签二维码,贴在产品包装上展示。
  • 谁关心:卖带能效标签商品(大家电 / 灯具 / 冰箱 / 显示器 / 电子屏幕 / 轮胎等)的卖家;亚马逊后台也要求上传能效标签图。
  • 什么时候管:上架前。没登记直接卖 → 罚款。详见 EPREL 数据库更新要求。

6. IOSS(Import One-Stop Shop,进口一站式申报机制)

  • 这是啥:从 2021 年 7 月开始的”小额进口简化申报”,商品价值 ≤ €150 的跨境 B2C 包裹,卖家 / 平台可以注册一个 IOSS 号,统一申报增值税,免去买家到货再缴税的麻烦。
  • 谁关心:从中国直发小包(€22-€150 之间)的卖家;亚马逊 Logistics / FBA 海外仓模式不适用(已缴过进口 VAT),主要是自发货 + 第三方海外仓发货。
  • 什么时候管:发货时。没 IOSS 号 → 买家到海关被卡,差评退货一起来。

7. PPWR(Packaging and Packaging Waste Regulation,包装与包装废弃物法规)

  • 这是啥:2026-08-12 全面执行的新包装法,替代原来的 94/62/EC 指令。要求包装物质含量(重金属 + PFAS)、最小化(空置容积 ≤ 50%)、标签(运营主体 + 批次追溯)、可回收等级(A/B/C)等多个维度全部合规。
  • 谁关心:所有在欧盟卖带包装商品的卖家,包括亚马逊 / TEMU / 速卖通都已发通知。
  • 什么时候管:2026-08-12 起强制。还有不到 14 个月窗口期,详见 PPWR 法规卡。

8. EUDR(EU Deforestation Regulation,欧盟反毁林法规)

  • 这是啥:2023 年通过的”反毁林”供应链法,要求牛 / 可可 / 咖啡 / 棕榈油 / 大豆 / 木材 / 橡胶 / 衍生品(皮革、巧克力、家具、橡胶轮胎…)在欧盟销售时,必须证明原料不是来自 2020 年 12 月 31 日之后被砍伐的森林。
  • 谁关心:卖上述原料及衍生品的卖家(巧克力、咖啡、皮革制品、橡胶制品、家具、纸制品…);需要做”地理坐标”尽职调查。
  • 什么时候管:2025-12-30 起对大企业执行,2026-06-30 对中小卖家执行(原计划)。细则仍在二级立法调整中,上产品前再核对最新动态。

你要做什么(按优先级)

第 1 优先级(第 1 周就做):

  • GPSR 欧代(全欧盟 1 份,通用)
  • CE 标志 + DOC(全欧盟通用)
  • 德国 LUCID 包装 EPR(德国站必备)
  • VAT(德国 FBA 入仓前)

第 2 优先级(第 2-6 周):

  • 法国 Citeo + 意大利 CONAI 包装 EPR
  • 法国 Triman 标识
  • WEEE 电气 EPR(卖电子产品)
  • EPREL(卖带能效标签商品)

第 3 优先级(关注中):

  • IOSS(自发货跨境包准备)
  • PPWR(2026-08 前的物质 / 包装 / 标签准备)
  • EUDR(看你产品是否在 8 类里)

常见问题

Q1:这 8 个是不是都要做? A:看你的产品 / 站点。GPSR + CE + 包装 EPR + VAT 是”基础四件套”,几乎所有卖家都要;其他看品类(EPREL、WEEE、IOSS、PPWR、EUDR)。建议照着上面优先级走。

Q2:欧代 / EPR 注册号亚马逊承认哪些服务商? A:亚马逊没有”白名单”,但认主流服务商(欧代如 ProductIP / Eurofins / mcs;EPR 如 stiftung ear / Citeo / CONAI / ECOLOGIC)。“野鸡”服务商(无欧盟实体办公室、价格低得不正常)要小心,审核可能被拒。

Q3:中文资料能直接翻译用吗? A:大方向可以,细则不行。CE DOC 模板、欧代协议模板网上能搜到中英文版,但涉及法规引用的部分建议找懂欧盟法的人过一遍。DOC 上签字的人是”自我声明”,错了要承担法律责任。

Q4:我看别的卖家啥都没做,链接还在,是他们不守规矩吗? A:可能原因很多:他们是”小批量试卖”(还没触发平台审核);商品在 GPSR 豁免清单(如纯服务、二手古董);或者他们也在等限流(只是还没轮到他)。别拿别人当参照,亚马逊的合规审核是分批次的,轮到你时,改已经来不及。

Q5:这 8 个名词之外,我还会撞到什么? A:还有几个中频出现的:CRA(网络弹性法,2027 年生效,电子产品要过网络安全测试)、ESPR(可持续产品法规,能效 / 标签的”上位法”)、PLD(产品责任指令修订)、Batteries Regulation(电池法规,2025-2027 分阶段执行)。新手先搞懂 8 个主力,这些先记名字,遇到再查。
