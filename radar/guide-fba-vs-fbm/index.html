一句话结论

新手无脑选 FBA。原因很简单:操作简单、Prime 标志自带流量、24 小时到货、亚马逊帮你处理客服和退货。FBM(自发货)留给大件、危险品、试款期 —— 99% 的新手期用不到。别在”自发货能省钱”的幻想里浪费 3 个月试错。

这是啥

亚马逊欧洲站的发货方式,就两种:

  • FBA(Fulfillment by Amazon,亚马逊物流):你把货提前发到亚马逊在德国/法国/意大利的仓库,客户下单后亚马逊自己打包、发货、客服、退货。你什么都不用管,只管进货和补货。
  • FBM(Fulfillment by Merchant,卖家自发货):客户下单后,你自己打包、自己找物流公司、寄到客户手里。客服、退货、纠纷,都是你自己处理。

看起来 FBM”省钱”(不用付 FBA 仓储费),实际是个坑:

  • 物流时效差(7-15 天到货,客户等不及 → 差评)
  • 没 Prime 标志(欧洲站 70% 客户优先选 Prime 商品)
  • 客服语言障碍(德国客户用德语问问题,你不会德语 = 亚马逊直接判你”响应慢” → 账户警告)
  • 退货成本高(FBM 退货买家寄回你国内地址,运费可能比产品还贵)

打个比方:FBA 像是”开品牌连锁店,亚马逊帮你管”——轻松、贵、流量大;FBM 像是”开独立小卖部,自己进货发货”——累、省、流量小。

为什么跟你有关

发货方式选错,3 个直接后果:

  1. listing 没流量:FBM 没 Prime 标志,欧洲站 60-70% 客户直接跳过 → 你的 listing 永远在搜索结果第 5 页之后
  2. 客服做到崩溃:FBM 你要自己处理退货、退款、纠纷,德国客户用法语投诉你不会回 → 账户绩效分数”Order Defect Rate(订单缺陷率)“飙高 → 封号风险
  3. 资金周转慢:FBM 客户收货才放款(7-15 天 + 退货期 14-30 天),资金回笼要 30-45 天;FBA 入仓后基本可以滚动补货,资金利用率高 50%

新店第 1-3 个月,你的”基础设施”—— listing 优化、合规、广告、运营——还没建好,千万别在发货方式上再加一层难度。FBA 是新手的”减负”工具,不是”烧钱”工具。

5 维对比表(FBA vs FBM)

维度FBA(亚马逊物流)FBM(自发货)
操作难度简单(发货入仓后什么都不用管)难(打包、贴单、联系物流、客服)
物流成本高(FBA 仓储费 + 配送费,小件 3-6 欧/件)低(发一单算一单),但退货成本极高
Prime 资格✅ 有(Prime 标志 + 24 小时到货)❌ 没有(欧洲站 70% 客户优先 Prime)
仓储费按月收(标准件 20-30 欧/m³/月,大件更贵)0(没库存压力)
客户体验24 小时到货,亚马逊客服,德/法/意/西语全支持看物流商(7-15 天),自己处理客服(语言坑)
现金流入仓后滚动补货,资金利用率高30-45 天回款,资金占用大
适合人群99% 的新手 + 稳定出单的老卖家大件 / 危险品 / 试款期 / 自有海外仓
隐性成本长期仓储费(>180 天加倍)退货国内运费 + 客户差评成本

一句话总结:FBA 卖”服务和省心”,FBM 卖”灵活性”。新手用钱换时间和确定性,老卖家用灵活性换利润。

何时选 FBA(99% 的情况)

符合以下任一条件,直接 FBA:

  • ✅ 你是新手,第一次做亚马逊欧洲站——别自己造轮子
  • ✅ 产品尺寸标准(≤ 45cm × 35cm × 20cm)——FBA 配送费不会贵到离谱
  • ✅ 产品重量轻(≤ 1kg)——FBA 费用可控
  • ✅ 产品没危险品(电池液体要看具体类目,部分能 FBA 部分不能)
  • ✅ 预期月销 ≥ 30 件——FBA 仓储费才划算
  • ✅ 追求 Prime 流量 + 客户体验——FBA 是唯一选择

举例:卖家居收纳盒、宠物用品、办公文具、健身小件、母婴日用品 —— 无脑 FBA。

何时选 FBM(少数情况)

以下情况,FBM 才有意义:

1. 大件 / 超重件(尺寸 > 60cm 或重量 > 5kg)

  • 家具、灯具、自行车配件、健身器材(跑步机、杠铃)
  • FBA 大件配送费贵到离谱(可能 15-30 欧/件),自发货找海外仓更划算

2. 危险品 / 限制品(亚马逊 FBA 不收)

  • 含电池的某些产品(FBA 要求 UN38.3 报告 + 危险品审核)
  • 液体、粉末、强磁(部分类目 FBA 直接拒收)
  • 易燃易爆、压力容器
  • 这种产品只能 FBM + 找专业危险品物流商

3. 试款期(新品前 50-100 件,不确定能不能卖)

  • 不要一上来就发 500 件到 FBA——卖不掉就是 6 个月仓储费
  • 试款期走 FBM + 国内直发,验证产品市场反应
  • 验证 OK 了再切 FBA——切换很快,Listing 不变

4. 定制 / 手工 / Made-to-Order

  • 个性化定制、刻字、按订单生产
  • 这种只能 FBM —— 客户下单才生产,不可能提前入仓

5. 已有海外仓(自建或第三方)

  • 德国 / 法国 / 波兰有自己或第三方的海外仓
  • 仓储成本 < FBA → 用 FBM + 海外仓本地配送,时效接近 FBA

绝大多数新手,99% 的情况都用 FBA。FBM 不是”省钱”,是”特殊场景的工具”。

3 步决策法(你到底选哪个)

步骤 1:看产品尺寸 + 重量

  • 尺寸 ≤ 45cm × 35cm × 20cm,重量 ≤ 1kg → FBA 无脑选
  • 尺寸 45-60cm 或重量 1-5kg → 算账:FBA 配送费 < 售价 20%?是:FBA;否:FBM
  • 尺寸 > 60cm 或重量 > 5kg → FBM + 海外仓

怎么算 FBA 费?在 Seller Central → Revenue Calculator(收入计算器)输入产品尺寸重量,直接看 FBA 配送 + 仓储总成本。

步骤 2:看你的资金

  • 有 3-5 万人民币压在 FBA 库存 → FBA
  • 只有 1-2 万,想先小批量试 → FBM 试款,3 个月后切 FBA
  • 完全没库存预算,纯想做 dropshipping(无货源) → FBM,但要接受”低流量 + 高退货率”

FBA 的资金逻辑:提前 1-2 个月备货 + 入仓 + 上架 + 跑广告,大概 3 个月才能回本。资金不够 3 个月周转,谨慎。

步骤 3:试用 3 个月 FBA 再说

不要一开始就”我永远用 FBA”或”我永远 FBM” —— 发货方式是动态调整的。

  • 第 1-3 个月:FBA(主力,跑通流程)
  • 第 4-6 个月:看哪些产品跑出来了,加量;哪些没跑出来,清仓 + 转 FBM
  • 第 6 个月后:对”主力产品”继续 FBA;对”边缘产品”切 FBM 减成本;对”季节性产品”清季末切 FBM

别死守一种方式。FBA + FBM 混用是成熟卖家的常态。

常见坑(新手 100% 会踩)

坑 1:“FBM 省钱,先做 FBM”

这是新手最常见的错误。表面上 FBM 不用仓储费,实际:

  • 没 Prime 流量(欧洲站客户直接跳过非 Prime)
  • 物流时效 7-15 天(德国客户等 3 天就投诉)
  • 自己处理客服(语言坑 + 退货坑)
  • 账户绩效分数”Order Defect Rate”容易爆表

结果:3 个月没出几单,listing 沉到搜索结果第 10 页,前面投入全部打水漂。99% 的新手 FBM 失败案例,根源都是”省小钱亏大钱”。

坑 2:发了 500 件到 FBA,结果 3 个月没卖出去

FBA 仓储费是双刃剑:

  • 6 个月内:正常收费(20-30 欧/m³/月)
  • 6-12 个月:加收长期仓储费(原费 × 1.5)
  • 超过 12 个月:加收超龄库存附加费(原费 × 3-5)

怎么避:

  • 首批入仓不超过 100-200 件
  • 3 个月没动销 → 立即清仓(降价 + 站外引流 + 多站点同步)
  • 别在 FBA 仓”养库存”,那是给亚马逊打工

坑 3:产品发到德国 FBA,法国客户下单从德国发,买家多等 5 天

FBA 是按”Pan-EU(泛欧计划)“或”EFN(欧洲物流网络)“区分的:

  • Pan-EU:亚马逊自动在多国分配库存,客户从最近的仓发货 → 快 + 贵
  • EFN:你只发到 1 国(比如德国),其他国客户下单从德国跨境发 → 慢 + 便宜

新手建议:

  • 前期 1 个国家(德国)→ 用 EFN 即可,单国发
  • 月销稳定 200+ 件 → 开 Pan-EU,让亚马逊自动分配库存
  • 别一上来就开 Pan-EU——你还没卖动,亚马逊帮你分到 5 国仓 = 5 国仓储费

关联:德国 / 法国 / 意大利,我先开哪个 —— FBA 仓布局跟着”先开哪个国家”走。

坑 4:危险品 / 电池产品硬塞 FBA

FBA 对很多品类有限制:

  • 带锂电池的产品(充电宝、电动牙刷、智能手表)→ 要 UN38.3 报告 + MSDS(化学品安全说明书),否则被拒收
  • 液体、粉末、压力容器 → 多数类目 FBA 不收
  • 易燃易爆 → 绝对不收

怎么避:进货前就问工厂”你的产品 FBA 能入仓吗”,让工厂提供危险品分类报告(MSDS / UN38.3)。不收就老老实实 FBM + 找专业危险品物流商。

坑 5:退货处理不当,FBA 退货 = 直接亏

FBA 退货,亚马逊不”免费销毁”:

  • 标准退货:亚马逊放回库存(可销售),但你的库存变成”used”(二手),影响后续销售
  • 客户退款:亚马逊先退给客户,再从你账户扣款——你可能钱货两空
  • 不可销售退货:亚马逊收费销毁(0.5-2 欧/件)

怎么避:

  • listing 描述精准(减少”与图片不符”退货)
  • 主图 + 详情页诚实(别夸张)
  • 尺寸 / 颜色信息写清楚(减少”买错”退货)
  • 退货率超过 5%,listing 健康度直接下滑

风险提示与免责声明

本站是信息整理和辅助判断工具,不构成物流策略、财务或商业建议。FBA 费用因产品尺寸、重量、类目、季节性(旺季 10-12 月仓储费翻倍)差异巨大,实际成本以 Seller Central 后台实时报价为准。发货方式选择请结合你的资金状况、产品特性、市场预期综合判断。FBA 不是”万能解”,FBM 也不是”省钱神器” —— 选错了都亏钱,选对了都赚钱。
