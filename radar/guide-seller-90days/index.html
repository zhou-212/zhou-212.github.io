一句话结论

90 天够你从 0 走到”基本合规”,不需要一次全做。把这 90 天拆成 5 个阶段(第 1 周 / 第 2-3 周 / 第 4-6 周 / 第 7-10 周 / 第 11-13 周),每周专心干 1-2 件事,就不会被法规压垮。

这是啥

你不是一个人在战斗:几乎所有”刚开欧洲站”的中国卖家,前 3 个月都是懵的 —— 谷歌搜一圈全是德语法语,亚马逊后台发邮件说”请补充 CE / GPSR / EPR”,一脸问号。

好消息是:欧盟合规有轻重缓急。有的事(比如 GPSR 责任人)不上产品就会被下架,优先级最高;有的事(比如 PPWR 物质检测)有 18 个月过渡期,先记着,慢慢补。

坏消息是:你不能”等以后再做”。亚马逊现在对 GPSR 缺失的产品,直接限流甚至下架;德国 LUCID 包装注册没有号,新批次根本进不了 FBA 仓。时间窗口是真实的。

为什么跟你有关

假设你 2026 年 7 月刚在 Amazon.de 上架了 20 个 SKU,9 月底之前会撞到这些事:

  • 亚马逊第 2 周开始发”EU Responsible Person 缺失”的邮件
  • 没有德国 LUCID 号 → 德国 FBA 入仓被卡
  • 法国 / 意大利客户开始看到你的产品,但你没注册 EPR → 当地税局 / 平台双重追讨
  • 卖电子 / 玩具类还要 CE 标识 + DOC 符合性声明

90 天不规划,5 个月后你可能一边补合规、一边看着 listing 一个个挂掉。

你要做什么(按周排活)

第 1 周:搞清”我要卖什么、要面对谁”

新手最容易犯的错:产品都上架了,才发现自己卖的蓝牙耳机要 CE、要 RED 无线电指令、要 EPREL、要 WEEE。先把 SKU 全部列出来,逐个问 3 个问题:

  1. 这个产品是什么品类?(电子 / 玩具 / 家居 / 服装 / 美妆 / 食品…)
  2. 有没有电池?有没有无线模块?(决定要不要额外的 CE 指令)
  3. 是全新还是二手?是不是翻新?(二手有部分豁免)

把答案填进一个 Excel 表,一列是 SKU,后面 5-6 列是合规需求。这张表你以后每加一个新产品都用得上。

关联:欧盟 GPSR 通用产品安全法规—— 先扫一眼,确认你的产品不在豁免清单里。

第 2-3 周:搞定”四件套”基础资料

不管你卖什么,以下 4 件事第 3 周之前必须做完:

  1. 欧盟责任人(Responsible Person):签一个欧代协议(几万到十几万人民币/年),拿到对方在欧盟的注册地址和邮箱。亚马逊后台”EU Responsible Person”字段填上,等审核。
  2. CE 标识 + DOC 符合性声明:产品本体 / 包装 / 说明书任选位置打 CE 标志(高度 ≥ 5mm,详见 CE 标识怎么打);DOC 是一份自我声明文件,写清楚你做了哪些测试、符合哪些指令。
  3. 德国 LUCID 包装注册:在 verpackungsregister.org 注册,拿到 LUCID 号,填到亚马逊后台。
  4. VAT 税号:如果你在德国 / 法国 / 意大利有库存,或者销售额超过当地远程销售起征点(目前 1 万欧元),必须申请 VAT。最快的办法是找当地税务代理。

关联:德国 LUCID 包装法、亚马逊欧洲站 8 大合规名词一次搞懂、新卖家开店基础资料检查清单

第 4-6 周:法国 + 意大利 EPR + 法国 Triman

德国搞定后,法国和意大利是下一站,主要是 EPR 三大类目(包装 / 电气 / 纺织品等):

  • 法国:到 ecologic-france.com / Citeo 等机构注册包装 EPR,拿到唯一识别符(Unique ID),打 Triman 标志。
  • 意大利:到 CONAI 注册包装 EPR,流程类似。
  • WEEE(电气电子):卖带电产品(充电器、电动牙刷、智能手表…)还要单独注册电气 EPR。

Triman 容易踩坑:它是法国特有的回收标志(一个垃圾桶 + 叉号),不带电池的普通产品要打 Triman,带电池的电子产品要打 Triman + 垃圾桶(打错了罚款)。详见 法国 EPR Triman 标识。

这一阶段可以稍微慢一点,但 6 周内要看到所有 EPR 注册号,不能拖。

第 7-10 周:补合规文档 + 准备追溯档案

这一阶段相对”轻”,主要是把前 6 周做的所有事,落成可追溯的文档:

  • 欧代协议(扫描件,存 5 年)
  • CE 测试报告(实验室出具的)
  • DOC 符合性声明(自己写,签字)
  • LUCID / Citeo / CONAI 注册号截图
  • 增值税申报记录

为什么重要:亚马逊、德国市场监管机构、法国 DGCCRF 任何一个来查,你 48 小时内要拿出来。提前归档,免得到时翻邮箱翻到崩溃。

关联:收到平台合规通知后的 48 小时应对清单

第 11-13 周:加新站点前,先复核一遍

到第 11 周,你的基础合规基本齐了。接下来如果想加西班牙 / 荷兰 / 瑞典 / 波兰,流程比德国简单(部分国家暂停或简化了 EPR),但别跳过验证步骤:

  • 老老实实打开亚马逊后台”Account Health” → “Compliance”
  • 每个新站点的”EU Responsible Person”字段都补全
  • 包装 EPR 在新国家补注册(部分国家已并入德国 LUCID)
  • 看一遍 PPWR 包装新规 2026-08-12 全面适用—— 这是下一个大动作,提前准备

第 13 周,给团队做一次”合规复盘”,更新那张 SKU 合规表。

常见问题

Q1:90 天来不及,只做前 4 周的事,行不行? A:短期能跑通,长期有风险。前 4 周搞定 GPSR + LUCID + CE,能撑住德国站不被下架;但法国 / 意大利 EPR 不做,3-6 个月内会被当地平台 / 税局找上。建议至少 6 周内把三国基础合规做齐,后 6 周用于补全。

Q2:我卖的是服饰 / 家居(不带电),需要做全部吗? A:可以省一点。不带电 → 不需要 WEEE 电气 EPR,不需要 RED 无线电指令;但 GPSR、LUCID、VAT、Triman(法国纺织品也要打)都躲不掉。具体看你 SKU 清单的那张表。

Q3:我找欧代 / 检测机构花了不少钱,有没有更便宜的路径? A:欧代市场已经透明化,几千到 1 万多人民币/年能搞定;但不要图便宜买来路不明的”野欧代”(可能是空壳公司,亚马逊审核会被打回)。检测报告找 SGS / TÜV / Intertek,价格虚高但报告含金量高,欧洲海关和亚马逊都认。

Q4:我能跳过哪个国家先不开? A:可以。亚马逊允许你只开德国站,先把德国做深;法国 / 意大利看你销量再决定。但开通前要合规做齐,不然一开通就被卡。详见 德国 / 法国 / 意大利,我先开哪个。

Q5:90 天之后,每个月要做什么? A:长期维护其实很轻:每月 1 次检查后台”Account Health”邮件;每季度复核一次 LUCID / EPR 号是否还在有效期(多数年费到期前 30 天邮件提醒);每年复检一次 CE / DOC(产品没变就不用重测)。其余时间专心卖货。
