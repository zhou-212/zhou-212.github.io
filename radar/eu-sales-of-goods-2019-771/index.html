EU 2019/771 售后保障指令:2025-07 起二手商品扩展适用,卖家责任 + 期限 + 应对

EU《商品销售指令》(Directive 2019/771, Sales of Goods Directive)在 2022-01-01 各成员国转化后,自 2025-07-30 起扩展适用至**二手 / 翻新商品**。新规统一欧盟全境售后保障规则,法定保质期 1-2 年(各国不同),卖家 1 年内负举证责任,买家有权选择**维修 / 替换 / 减价 / 终止合同**。基于 EUR-Lex 法规原文 + 欧盟委员会公告,梳理卖家责任 + 时间表 + 5 步合规指南。

一句话结论

欧盟《商品销售指令》(EU) 2019/771 在 2022-01-01 各国转化后,自 2025-07-30 起扩展适用至二手 / 翻新商品。所有在欧盟销售的实物商品(亚马逊、独立站、二手平台)都有 1-2 年法定售后保质期(各国不同),卖家 1 年内负举证责任。买家有权选择维修 / 替换 / 减价 / 终止合同。罚款 1000-10000 €/案件(看国家)。

这是啥

EU 2019/771 售后保障指令(Sales of Goods Directive)是欧盟消费者保护框架的核心法规,2022-01-01 起在所有成员国转化后生效,2025-07-30 起扩展适用至二手 / 翻新商品(各成员国转化截止日)。

核心规则:

  • 法定保质期 1-2 年(各成员国不同:德国 1 年 / 法国 2 年 / 荷兰 2 年 / 意大利 2 年 / 西班牙 3 年)
  • 卖家对瑕疵承担 1 年举证责任(证明问题不是你的责任)
  • 买家有权选择:维修 → 替换 → 减价 → 终止合同(优先级递减)
  • 二手商品可约定缩短保质期(不能短于 1 年)
  • 商业保证(warranty)不能替代法定保障

适用 vs 不适用

适用:

  • ✅ 所有实物商品(服装、电子、家具、家居、玩具、运动器材等)
  • ✅ 数字内容(若与商品结合)
  • ✅ 二手 / 翻新商品(2025-07-30 起)
  • ✅ B2C 销售(消费者对商家)
  • ✅ 跨境销售(通过电商平台)

不适用:

  • ❌ B2B 销售(商家对商家)
  • ❌ 服务类(维修、咨询)
  • ❌ 纯数字内容(电子出版物、App 内购买)
  • ❌ 拍卖(特定情况)
  • ❌ 古董 / 收藏品(卖家明确声明)

为什么跟你有关

亚马逊 EU 平台规则(Seller Central 2025-06):

“All sellers on Amazon EU must comply with Directive (EU) 2019/771 on the sales of goods, including the extension to second-hand goods as of July 30, 2025. Sellers must provide clear return policy, warranty terms, and A-to-Z claim handling.”

没做的后果:

  • 买家通过 A-to-Z Guarantee 索赔 → 卖家必须退款
  • 多次索赔 → 账户健康度(ODR)下降
  • 欧盟消费者保护机构(BVL / DGCCRF / AGCM)处罚
  • 国家层面罚款:1000-10000 €/案件
  • 集体诉讼风险(欧洲消费者组织 BEUC 推动)

特别提醒:

  • 即使你的亚马逊后台 return policy 写了”30 天无理由退货”,也不能限制买家的法定 1-2 年保质期
  • 即使你的 listing 写了”as is”或”final sale”,也不能豁免法定保障
  • 即使产品经过”质检”或”测试”,也不自动免责

卖家核心义务

1. 法定保质期(1-2 年)

  • 期间内,商品不符合合同(有瑕疵)即触发卖家责任
  • 不符合合同包括:损坏 / 故障 / 数量不对 / 型号不对 / 描述不符 / 用途不符 / 二手商品与声明状况不符
  • 卖家必须维修 / 替换 / 减价 / 退全款(按买家选择)

2. 1 年举证倒置

  • 售后保质期 1 年内,卖家必须证明问题不是你的责任
  • 1 年后,买家必须证明问题从一开始就有
  • 实际操作:保存产品质检报告 / 物流凭证 / 出厂记录

3. 4 步补救优先级

买家有选择权,但卖家可以先尝试维修(若合理且不构成”重大不便”):

第 1 步:维修(Repair)      - 卖家主动,无重大不便
第 2 步:替换(Replacement) - 维修失败 / 不可能 / 不合理
第 3 步:减价(Price Reduction) - 瑕疵轻微,买家接受
第 4 步:终止合同(termination) - 瑕疵严重,退款退货

4. 商业保证(Commercial Guarantee / Warranty)不能替代法定保障

  • 卖家可以提供额外商业保证(如”3 年保修”)
  • 但不能缩短或替代法定保质期
  • 商业保证必须明确:覆盖范围 / 期限 / 条件 / 申请流程
  • 商业保证对买家免费

5. 二手 / 翻新商品特殊规则(2025-07-30 起)

  • 法定保质期仍适用,但可约定缩短(不能短于 1 年)
  • 卖家必须明确说明二手商品的状况(成色、功能、缺陷)
  • 卖家在 1 年内仍负举证责任(除非证明买家已知道瑕疵)
  • listing 必须标注”As new / Refurbished / Used - Good / Used - Acceptable”等成色

5 步合规指南

第 1 步:产品 listing 描述规范化(预算 0 €,1 周)

做什么:

  • 每条 SKU 详情页必须详细描述产品状况
  • 全新商品:功能 / 规格 / 包装 / 配件 / 尺寸 / 重量 / 颜色
  • 二手商品:成色等级 / 已知瑕疵 / 包装情况 / 是否原厂配件
  • 翻新商品:翻新过程 / 配件更换 / 保修范围

工具:

  • 亚马逊后台 Inventory → Edit Listing
  • 字段:Title / Bullet Points / Description / Key Features

避坑:

  • 别写”as is”或”final sale”试图免责 —— 法定保障不可豁免
  • 别隐瞒瑕疵 —— 故意隐瞒瑕疵 = 欺诈
  • 别用过度的免责声明(消费者法院通常不认可)

第 2 步:Return Policy + Warranty Policy 写清楚(预算 0 €,1 周)

亚马逊后台:

  • Settings → Return Policy:必须符合目标国法定保质期
    • 德国站:写”1 year legal warranty”或更长
    • 法国站:写”2 years legal warranty”或更长
    • 西班牙站:写”3 years legal warranty”或更长
  • Settings → Marketplace Settings:各国 warranty 政策
  • 提供清晰退出流程:买家怎么申请、给多少时间、谁出运费

关键内容:

  • 法定保质期 1-2 年(根据国家)
  • 二手商品成色说明
  • 商业保证(若提供)的具体范围
  • 申请流程 + 联系方式

第 3 步:维修 / 替换网络搭建(预算 0-2000 €/SKU/年)

目标:

  • 7-14 天内完成维修(若商品可修)
  • 30 天内完成替换(若商品可替)

方案:

  • 小卖家:与本地维修店合作(0-500 €/SKU/年)
  • 中型卖家:在欧盟核心国(德/法/意/西)建立维修中心
  • 大卖家:自建欧洲维修中心 + 配件库存(2000-10000 €/SKU/年)
  • 响应超期:买家有权要求直接换新 / 退款(不再给维修机会)

维修合同必备:

  • 维修时效(SLA)
  • 维修失败的处理(直接换新)
  • 配件保修(通常 90 天)
  • 维修记录保存(5 年)

第 4 步:1 年举证证据管理(预算 0-500 €,持续)

做什么:

  • 出厂质检报告 + 批次记录
  • 入库 / 出库照片 / 视频
  • FBA 接收扫描记录
  • 包装防损措施(护角 / 缓冲 / 防水)
  • 物流温度记录(易腐品)
  • 售后维修记录(类似问题频率分析)

工具:

  • 内部 ERP / WMS 系统
  • 亚马逊库存报告(FBA Inventory Reports)
  • 厂商出厂合格证 + 测试报告

关键:

  • 证据保留至少 5 年(欧盟消费者保护机构可能追溯)
  • 1 年内卖家主动举证(不是被动等买家提供)

第 5 步:争议处理流程 + 法律支持(预算 500-2000 €/年)

亚马逊平台争议:

  • A-to-Z Guarantee 索赔 → 3 天内响应
  • Return Request → 1 天内响应
  • 多次索赔 → 申诉(提供证据)
  • 平台仲裁 → 通常对买家有利,卖家败诉率高

欧盟法院 / 消费者组织:

  • 欧洲消费者组织 BEUC 推动集体诉讼
  • 各国消费者保护机构 + 法院
  • 跨境争议:European Consumer Centre(ECC-Net)

预防措施:

  • 售前:listing 描述准确 + 详尽
  • 售中:快速响应买家咨询
  • 售后:主动提供维修 / 替换,避免升级
  • 复盘:每季度分析纠纷原因,改进产品 / 包装

4 个常见误区(基于公开报告)

误区 1:“Return Policy 写 30 天 = 法定 30 天” 错。亚马逊平台 return policy 是无理由退货(消费者保护),不能缩短法定保质期。法定保质期是 1-2 年,跟 return policy 无关。

误区 2:“二手商品可以不保” 错。2025-07-30 起 二手 / 翻新商品法定保质期仍适用,可约定缩短(不能短于 1 年)。listing 写”Used - As is”不豁免法定责任。

误区 3:“商业 3 年保修 = 法定 3 年” 错。商业保修是额外服务,不替代法定保障。法定保质期仍按 1-2 年走(看国家)。两个独立体系,都要做。

误区 4:“A-to-Z 索赔败诉没关系” 错。多次 A-to-Z 败诉 → 账户健康度 ODR 上升 → 账户审查 / 暂停。提前做好 listing 描述 + 售后响应,避免败诉。

何时做 / 何时不做

符合以下任一条件,立刻做:

  • ✅ 你在 Amazon EU 销售实物商品
  • ✅ 你的商品在目标国法定保质期(1-2 年)内
  • ✅ 你卖二手 / 翻新商品(2025-07-30 起)
  • ✅ 你重视 listing 长期稳定性 + 账户健康

暂缓:

  • ⚠️ 你只卖服务(无实物)
  • ⚠️ 你只做 B2B(消费者保护不适用)
  • ⚠️ 你只卖古董 / 收藏品(明确声明可豁免)

FAQ

法定保质期 1 年 vs 2 年怎么选?

按目标销售国的法定保质期:

  • 1 年:德国、瑞典(部分类目)
  • 2 年:法国、意大利、荷兰、比利时、奥地利、波兰、捷克、罗马尼亚、匈牙利、希腊、丹麦、葡萄牙
  • 3 年:西班牙、瑞典(部分类目)
  • 建议统一写 2 年(覆盖大部分国家)

二手商品缩短保质期怎么算?

可以缩短,但不能短于 1 年。例如:法国站 2 年法定,可以约定”二手商品 1 年保质期”,listing 明确说明。

1 年举证责任什么意思?

商品交付后 1 年内出现瑕疵,卖家必须证明问题不是你的责任(例如:买家使用不当 / 物流损坏 / 自然损耗)。1 年后,买家必须证明问题从一开始就有。

商业保修(commercial guarantee)是什么?

商家自愿提供的额外保修服务,如”3 年保修”、“全球联保”。不能替代法定保质期。卖家可以单方面决定保修范围 + 期限,但必须明示。

卖家在欧盟有公司,是否影响保质期?

不影响。法定保质期是消费者保护规则,适用所有 B2C 销售,与卖家注册地无关。中国公司卖到欧盟的 Amazon EU,法定保质期仍然 1-2 年。

集体诉讼 / 集体索赔怎么办?

欧洲消费者组织 BEUC 推动消费者集体诉讼,目前在德国、法国、葡萄牙已立法,其他成员国陆续推动。多买家同一瑕疵可能被集体索赔,建议提前合规避免集体诉讼风险。

亚马逊 A-to-Z 索赔败诉影响多大?

A-to-Z 索赔败诉 → ODR(Order Defect Rate)上升 → 账户健康度下降 → 销售权限受限 / 账户暂停。ODR > 1% 触发账户审查。多次败诉 + 金额大 → 直接暂停。
