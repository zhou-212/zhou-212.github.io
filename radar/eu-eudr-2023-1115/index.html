欧盟 EUDR 反毁林法规 已生效:2025-12-30 大企业全面适用,亚马逊对 7 类商品要求 DDS

欧盟《反毁林法规》(EU) 2023/1115 于 2023-06-29 生效,经 2024/3234 修订,大型 / 中型经营者适用日期为 2025-12-30,微小企业延后至 2026-06-30。覆盖牛肉 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木材及衍生品,要求尽职调查声明 DDS + 地块地理坐标 + 不毁林证明(2020-12-31 cutoff)。亚马逊对 7 类商品已发布公告。

一句话结论

如果你的产品是木材 / 棕榈油 / 橡胶 / 可可 / 咖啡 / 大豆 / 牛肉及衍生品(皮革 / 巧克力 / 纸 / 家具 / 印刷品等),2025-12-30 起大企业适用、2026-06-30 起微小企业适用 —— 卖家必须逐批次上传尽职调查声明 DDS(EU 信息系统,前称 TRACES NT)+ 地块地理坐标(geolocation polygons)+ 不毁林证明(2020-12-31 后未毁林)。亚马逊已发布公告,7 类商品分阶段被要求 DDS。

适用范围

  • 法规级别:欧盟 Regulation(直接适用全境)
  • 覆盖范围:7 类商品及其衍生品(Article 3 + Annex I)
  • 7 类原料商品:牛肉(cattle)、可可(cocoa)、咖啡(coffee)、棕榈油(palm oil)、橡胶(rubber)、大豆(soya)、木材(wood)
  • 典型衍生品:
    • 木材 → 木制品、家具、纸浆、纸、印刷品、木炭、胶合板、刨花板
    • 棕榈油 → 食用油、生物燃料、化妆品原料
    • 橡胶 → 轮胎、橡胶手套、密封件
    • 大豆 → 饲料、生物燃料、大豆油
    • 可可 → 巧克力、糖果、可可脂 / 粉
    • 咖啡 → 烘焙 / 速溶 / 胶囊
    • 牛肉 → 皮革、明胶、牛肉副产品
  • 不适用:Article 1(2) 例外 —— 已加工到 end-of-life / 循环利用 / 回收材料的产品有部分豁免

关键时间线

  • 2023-06-09:Regulation (EU) 2023/1115 在 Official Journal 发布
  • 2023-06-29:生效(entry into force)
  • 2020-12-31:毁林 cutoff 日期:产品在 2020-12-31 之后不得有毁林行为
  • 2024-09-19:Regulation (EU) 2024/3234 修订:延期原定 2024-12-30 适用日期
  • 2025-12-30:大型 / 中型经营者开始适用 EUDR 主体义务(亚马逊公告的 deadline)
  • 2026-06-30:小型 / 微型企业开始适用
  • 2028-06-30:首次全面评估:欧盟委员会评估法规成效,可能扩列

核心义务 —— 经营者 / 贸易商

EUDR 区分经营者(operator)、大型贸易商 / 中型贸易商两类主体,义务递减:

  • 经营者(将 EUDR 商品首次投放欧盟市场 + 出口):完整 3 步骤 DDS
  • 大型贸易商(已投放欧盟市场后的销售):同样需要 3 步骤 DDS
  • 小型贸易商(员工 < 250 人 + 营收 ≤ 4000 万欧元):简化 DDS
  • 微小企业(员工 < 10 人 + 营收 ≤ 200 万欧元):只保留”可追溯性记录”

亚马逊卖家最常见身份:经营者(own brand)或大型贸易商(reseller),均需完整 DDS。

核心义务 —— 三步骤尽职调查(Due Diligence)

第 1 步:信息收集(Article 9)

  • 商品描述(HS code、品类、规格、产地)
  • 生产/养殖地块的地理坐标(geolocation,polygon 形式):农业用地多边形 ≤ 4 公顷(理想 ≤ 0.5 公顷)
  • 供应商名称、地址、联系方式
  • 生产国、生产地区(province/district)
  • 是否符合当地法律(legality)
  • 不毁林证明(satellite 影像 / 第三方认证 / 政府数据)

第 2 步:风险评估(Article 10)

  • 国家 + 地区 + 产地三级的毁林风险评估
  • 供应链尽职调查(是否存在混源 / 不明来源)
  • 与第三方认证(FSC / PEFC / RSPO / RTRS / 4C 等)交叉核对

第 3 步:风险缓解(Article 11)

  • 高风险商品:额外的独立第三方审计
  • 与供应商签订合规采购承诺(contractual clauses)
  • 暂停 / 退出高风险供应链关系
  • 持续监测毁林风险

提交与监管:EIS 系统

  • EU Information System(前称 TRACES NT,2024 年底升级):卖家逐批次在系统中提交 DDS Reference Number
  • 提交方:欧盟境内的首次经营者(operateur,可以是进口商 / 品牌方 / 本土代表)
  • 非欧盟卖家(如中国 / 美国品牌方):通常通过欧盟授权代表(Authorised Representative)代为提交
  • 海关凭 DDS Reference Number 放行,无编号的批次会被扣留 / 退回
  • 数据保留 5 年

处罚与执法

  • 罚款:最高成员国年度营业额 4%(按 EU 营业额计)+ 没收产品 / 收益
  • 经营许可限制:严重违规者临时吊销在欧盟的经营资格
  • 追溯时效:违规行为的罚款时效为 5 年
  • 政府采购黑名单:违反 EUDR 的实体在 3 年内可能无法参与欧盟公共采购

你现在要做什么(按商品类)

直接命中 7 类原料的卖家(木材 / 棕榈油 / 橡胶 / 可可 / 咖啡 / 大豆 / 牛肉):

  1. 2025-10 前完成 EUDR 信息收集模板准备,生产地块多边形数据
  2. 供应商对接:让农场 / 种植园 / 牧场 / 林场提供地块坐标 + 砍伐 / 种植历史 + 第三方认证(FSC / PEFC / RSPO / 4C 等)
  3. 风险评估 + 缓解文件:建立内部 EUDR 合规手册
  4. EU Information System 注册:通过欧盟授权代表或直接进口商完成(非欧盟卖家必须通过 AR)
  5. 持续监测:每批货的 DDS 独立提交(同商品不同批次需独立提交)

衍生品卖家(皮革 / 巧克力 / 印刷品 / 家具 / 轮胎等):

  1. 追溯原料溯源:对每批原料(皮革、棕榈油衍生物、橡胶原料)收集供应商 DDS 副本
  2. 下游再加工:最终组装环节要求前序加工商提供 EUDR 合规声明
  3. 记录保留 5 年:含原料来源、生产日期、DDS 引用

亚马逊相关动作

  • 2025 年起,亚马逊对 EUDR 管制商品的卖家逐步要求提供 DDS Reference Number
  • 亚马逊通过 Seller Central Product Compliance 入口上传 / 验证 DDS
  • 7 类商品 ASIN 将分阶段被锁定直至 DDS 上传成功
  • 2025-Q3 起已观察到亚马逊主动给 EUDR 管制品卖家发送合规通知邮件

合规清单(自检 5 项)

  • 产品是否命中 7 类原料 / 衍生品清单(牛肉 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木材 及皮革 / 巧克力 / 纸 / 家具 / 轮胎 等)
  • 生产地块地理坐标已采集(polygon 形式)
  • 供应商提供不毁林证明 + 第三方认证(FSC / PEFC / RSPO / 4C 等)
  • EU Information System DDS 已注册(通过欧盟授权代表 AR 或直接进口商)
  • 5 年数据保留机制已建立(原料来源、生产日期、DDS 引用)

不处理的潜在影响

  • 高概率:2025-12-30 后,未提交 DDS 的 EUDR 商品在欧盟海关一律扣留
  • 高概率:亚马逊对未上传 DDS 的 ASIN 限流 / 下架
  • 可能:成员国市场监管机构罚款(最高 EU 营业额 4%)+ 没收产品
  • 可能:3 年欧盟公共采购黑名单
  • 明确:本卡片不构成法律意见;具体商品类的 HS code 归类、衍生品范围、polygon 精度要求,需查 EUDR 二级立法及欧盟委员会 FAQ

常见误区

  • 误区 1:“我卖的是包装纸,不是木材” —— 错。纸浆、纸、印刷品都是木材的核心衍生品。
  • 误区 2:“我卖的是巧克力,不是可可豆” —— 错。可可脂 / 可可粉 / 巧克力都是 EUDR 衍生品。
  • 误区 3:“FSC / PEFC 认证 = EUDR 合规” —— 部分对。FSC / PEFC 是第 1 步(信息收集)证据,但不替代第 2 步(风险评估)+ 第 3 步(风险缓解)+ 实际 DDS 提交。
  • 误区 4:“我是亚马逊 FBA,平台替我处理” —— 不一定。FBA 物流 ≠ EUDR 合规;卖家仍需自行完成 DDS。
  • 误区 5:“我不在欧盟,EUDR 不管我” —— 错。EUDR 是产品法规,任何”向欧盟投放 EUDR 商品”的行为都受其约束。
