欧盟 ESPR 生态设计法规 已生效:数字产品护照 (DPP) 分阶段落地,几乎所有物理产品受波及

欧盟《可持续产品生态设计法规》(EU) 2024/1781 于 2024-07-18 生效,2025-2027 按产品类分阶段开始执行,首次把'数字产品护照 DPP + 可修复性指数 + 再生原料 + 碳足迹'写进硬性义务,几乎覆盖所有物理产品(食品 / 药品 / 军火明确豁免)。是 PPWR / Batteries / 纺织品 DPP 的母法,亚马逊欧洲站卖家要按品类提前布局。

一句话结论

ESPR (EU) 2024/1781 是 PPWR / 电池护照 / 纺织品 DPP 的母法,2024-07-18 生效,首批产品类(纺织品 / 电子产品 / 家具 / 床垫 / 轮胎 / 洗涤剂 / 油漆 / 钢铁)的数字产品护照 DPP在 2025-2027 分阶段强制 —— 卖家要按品类提前 12-18 个月规划。覆盖几乎所有物理产品(食品 / 药品 / 军火明确豁免),处罚由各成员国设定,必须”effective, proportionate and dissuasive”。

适用范围

  • 法规级别:欧盟 Regulation(直接适用全境)
  • 覆盖范围:所有在欧盟市场投放或投入使用的物理产品(physical goods)
  • 替代关系:ESPR 取代原 Directive 2009/125/EC(Ecodesign 生态设计指令)+ 部分内容与 (EU) 2024/825 联合协调
  • 典型适用产品(目前已知首批 / 计划优先规制品类):
    • 纺织品与服装
    • 电子产品与电气设备(具体类目待二级立法)
    • 家具与床垫
    • 轮胎
    • 洗涤剂与清洁产品
    • 油漆与涂料
    • 钢铁与建材
    • 中间产品(化工、塑料、橡胶制品)
  • 明确豁免(Article 2):
    • 食品、饲料、药品、医疗器械、人用/兽用产品
    • 公路、铁路、水路、航空运输工具(客车 / 货车 / 船舶 / 飞机)
    • 军用 / 国防 / 航天产品
    • 文化遗产保护产品(古董)
    • 种子、植物等活体产品

关键时间线

  • 2024-06-28:Regulation (EU) 2024/1781 在 Official Journal (L 2024/1781) 正式发布
  • 2024-07-18:生效(entry into force)
  • 2025-04-18:首个 ESPR 工作计划(working plan) 由欧盟委员会采纳
  • 2025-07-18:ESPR 主体条款开始适用(18 个月过渡期)
  • 2026-2027:首批授权立法(delegated acts) + 数字产品护照 DPP 在具体产品类上强制
  • 2030+:覆盖更多产品类,各产品类分别有 18 个月过渡期
  • 注:每个产品类的具体时间表由单独的 Delegated Regulation 决定,首批 DPP 强制时间多在 2025-2028 之间

核心义务 —— 产品性能要求框架

ESPR 在 Annex I 列出了通用性能要求框架,具体阈值由授权立法(delegated acts)按产品类细化:

  • 耐用性(durability):产品预期寿命的最低标准
  • 可修复性(reparability / repair index):可维修性评分 + 易获取的维修信息
  • 可回收性(recyclability):产品报废后可回收材料占比
  • 再生原料含量(recycled content):强制最低再生原料比例
  • 碳足迹与环境影响足迹(carbon & environmental footprint):基于 PEF 方法(与 PPWR 协同)
  • 物质限制:有害物质管控(与 REACH 协同)
  • 能效与水资源效率:能源 / 水消耗最低标准
  • 可持续供应链:供应链尽职调查
  • 产品信息:产品标签、说明书、合规信息的最低要求

数字产品护照 DPP (Digital Product Passport) 详解

  • 形式:每件产品配 DPP 标识(二维码 / RFID / NFC 标签),扫码即获取产品合规数据
  • 数据范围:碳足迹、再生原料、可修复性指数、制造商信息、CE 标志、维修指南、回收路径
  • 数据载体:DPP 数据由欧盟中央注册系统统一管理;访问方包括消费者、监管机构、维修商、回收商
  • 强制时间:按产品类分阶段,首批 2025-2027 强制(具体看二级立法)
  • 重要前置:电池护照(Digital Battery Passport)是 DPP 的特例,由 EU 2023/1542 单独规定;纺织品 DPP 是 ESPR 首批强制项之一
  • 与 PPWR 关系:PPWR 包装部分也要求”包装 + 产品”协同的 DPP 数据结构

合规清单(自检 5 项)

  • 产品是否豁免确认:食品 / 药品 / 军用 / 公路铁路水运航空运输工具 → 暂时不直接受 ESPR 约束;其余物理产品均受波及
  • 产品类 DPP 时间表已确认:订阅欧盟委员会 ESPR 工作计划,看自己品类首批 / 后续批次时间
  • OEM 已能提供合规数据:可修复性 / 耐用性 / 再生原料 / 碳足迹
  • DPP 数据采集流程已搭建:成分、原料来源、维修手册、回收路径
  • PPWR 包装协同准备:同时满足包装法规的 DPP 数据结构与物质限制

你现在要做什么(按产品类)

所有品类卖家(必做):

  1. 盘点产品是否在豁免清单内。食品 / 药品 / 军用 / 运输工具 → 暂时不直接受 ESPR 约束,但衍生 / 配件 / 包装可能间接受影响
  2. 订阅欧盟委员会 ESPR 工作计划。首批产品类在 2025-04 公布,官方入口:https://ec.europa.eu/info/energy-climate-change-environment/standards-tools-and-databases/products-mechanisms_en
  3. 关注产品类 Delegated Regulation 时间表。一旦某类产品被纳入,会有 18 个月过渡期

首批优先品类(纺织品 / 电子 / 家具等)卖家(快做):

  1. OEM 协同:让供应商准备可修复性 / 耐用性 / 再生原料数据
  2. 建立 DPP 数据采集流程:产品成分、原料来源、维修手册、回收路径
  3. DPP 系统对接:选 DPP 数据服务方(目前有 Cirpass / DPP Lab / 自建)
  4. 包装协同:同时满足 PPWR 的包装 + DPP 数据结构
  5. 碳足迹基础工作:PEF 方法学(与 PPWR / 电池护照同基线)

亚马逊相关动作

  • 亚马逊欧洲站目前未对 ESPR 单独建合规门户(暂未要求上传 DPP),但包装标签受 PPWR 影响
  • 亚马逊可能成为DPP 数据消费者之一(平台需要展示合规数据),提前与亚马逊客户经理沟通预期
  • 纺织品 / 电子产品等大品类可能先被亚马逊纳入”产品合规高优先级”
  • Listing 商品描述建议加注”DPP 准备中”或”碳足迹公开”等透明化信号

不处理的潜在影响

  • 可能:被纳入首批产品类后,欧盟海关对未配备 DPP 的产品扣留
  • 可能:成员国市场监管机构罚款(各成员国设定阈值,但 ESPR 要求”有效、相称、有威慑力”)
  • 可能:亚马逊对未合规的纺织品 / 电子产品限流(平台规则 + 法规要求双重作用)
  • 明确:本卡片不构成法律意见;具体产品类的 DPP 时间表、性能阈值、再生原料比例,需查具体的 Delegated Regulation

常见误区

  • 误区 1:“我卖的不是纺织品,ESPR 不适用” —— 错。ESPR 覆盖几乎所有物理产品,首批优先纺织品 / 电子 / 家具,后续扩展到钢铁 / 化工 / 建材 / 中间产品,任何欧盟卖家早晚受影响。
  • 误区 2:“DPP 是 5 年后才强制的事” —— 错。纺织品 / 电子产品 DPP 在 2025-2027 期间分阶段强制,OEM / 供应商对接至少要 12-18 个月才能完成。
  • 误区 3:“DPP 等同于产品溯源” —— 部分对但不全。DPP 是合规数据 + 碳足迹 + 可修复性 + 维修信息,远超过传统溯源的范围。
  • 误区 4:“ESPR 和 PPWR 是一回事” —— 不。ESPR 主管产品本身(产品设计 + 性能);PPWR 主管包装(包装材料 + 物质)。两者协同,数据可互相引用。
