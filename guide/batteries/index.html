欧盟电池与废电池法规 入门指南

只要产品含电池就触发:碳足迹、电池护照、再生原料、回收效率。

欧盟《电池与废电池法规》(EU 2023/1542) 已经 2025-08-18 全面适用,替代原 2006/66/EC 电池指令。新规覆盖**碳足迹、再生原料、回收效率、电池护照、便携式 / LMT / SLI / 工业 / EV 五大类电池**,亚马逊已对在欧盟销售的含电池商品(充电宝、玩具、电子产品、电动工具)启动合规管控。重点:这条规则覆盖**5 类电池**,不是只针对"纯电池"卖家 ——**只要产品内含电池**,就触发要求。本指南把电池法规的所有内容汇总到一起,按"是什么 / 谁要遵守 / 现在做什么"三段读,大概 12 分钟心里有底。

如何行动(5 步)

  1. 1

    识别你的产品含哪类电池

    5 大类:便携式(纽扣 / 5 号 / 7 号等)、LMT(轻型运输工具,如电动滑板车电池)、SLI(汽车启动电池)、工业电池(> 2.5 kWh 工业用途)、EV(电动汽车)。绝大多数中国卖家做的是"便携式"或"LMT"。

  2. 2

    完成电池标签 + 化学成分披露

    2025-08-18 起强制:容量(mAh / Wh)、化学成分(Li-ion / Ni-MH / Lead-acid 等)、回收符号(垃圾桶打叉)、二维码(可访问技术文档)。找包装印刷厂加新版标签,或贴"补充标签"覆盖。

  3. 3

    注册生产者责任(电池回收)

    每个成员国都有电池回收系统:德国 ear / 法国 Corepile / 意大利 COBAT 等。注册后拿到唯一识别符,跟 EPR 包装注册(LUCID)分开。Amazon 后台"Manage your EPR compliance" 需分别填写。

  4. 4

    准备碳足迹 + 再生原料披露

    2025-2027 分阶段要求:碳足迹声明(CFP)、再生原料含量(钴 / 锂 / 镍 / 铅)。供应链向上追溯到矿产 / 冶炼厂,需要供应商配合。

  5. 5

    关注 2027-02 起的电池护照(Battery Passport)

    LMT / 工业 / EV 电池 2027-02-18 起贴"电池护照"二维码,接入欧盟统一数据库,披露:容量、健康状态、生产日期、化学成分、供应链信息。便携式电池暂不要求,但建议提前准备。

常见问题

Q 我用 5 号干电池供电的玩具,算"含电池商品"吗?
A

算。不管是干电池、纽扣电池、锂电池,只要产品本体带电池或需要电池才能工作,都属于覆盖范围。

Q 电池标签是印在电池上,还是印在产品包装上?
A

都印。电池本体印化学成分 / 容量 / 回收符号;产品包装上印"含电池"标识(适用指令图标)和"如何取出电池"说明。Amazon 后台也需在合规页面填写电池信息。

Q 电池护照是必须要做的吗?什么时候?
A

便携式电池(纽扣 / 5 号 / 7 号等)暂不要求电池护照。LMT(电动滑板车 / 电动车)、工业电池、EV 电池,2027-02-18 起强制。所有在欧盟销售 / 投放市场的电池都要进入数据库登记。

Q 我已经注册了 LUCID 包装 EPR,电池 EPR 要单独注册吗?
A

要单独注册。LUCID 是包装回收,电池 EPR 是电池回收,两个不同的体系,注册号也是分开的。Amazon 后台"Manage your EPR compliance" 需分别填写。
