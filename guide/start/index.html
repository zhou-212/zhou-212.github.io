如何行动(5 步)

  1. 1

    先看 90 天路线图(知道 0-90 天该做啥)

    90 天拆 5 个阶段:第 1 周(开店 + 选品 + VAT)→ 第 2 周(德国 LUCID + 法国 EPR)→ 第 3 周(GPSR 责任人 + 多语言说明书)→ 第 4 周(CE + EPREL + 备份)。每周专心 1-2 件事,不会被法规压垮。

  2. 2

    搞懂 8 个最常见名词(破冰)

    GPSR(产品安全 + 欧代)、CE(合格证)、EPR(生产者责任)、VAT(增值税)、EPREL(能效数据库)、IOSS(进口申报)、PPWR(新包装法)、EUDR(反毁林)。这 8 个搞懂,80% 的新手疑问能解决。

  3. 3

    选第一站(德国优先,后两国顺水推舟)

    德国站流量最大、合规最严,但搞定后法国 / 意大利 80% 是重复工作。先开德国,法国意大利按"水往低处流"推;反之先开法国或意大利,德国那一套 LUCID + GPSR + CE 还是要补,等于做两遍。

  4. 4

    收到 GPSR / 亚马逊合规邮件怎么回(3 套中英模板)

    80% 情况:在后台"EU Responsible Person"字段填好欧代,等系统抓取,邮件自动消失;15% 情况:回邮件提供欧代协议 + 公司注册证明;5% 情况:被抽审多个 ASIN,需要按模板提供批量证明。

  5. 5

    建 1 个自动备份的合规文件夹

    云盘(Google Drive / OneDrive / 飞书云文档)建一个"EU 合规"文件夹,把 CE / DoC / 测试报告 / VAT 证书 / LUCID 证书 / EPR UID / GPSR 责任人都放进去,开自动同步。平台 / 海关 / 监管机构发合规通知时,要求 48 小时内提交文件,找不到 = 直接下架。

常见问题

Q 我是个人卖家,公司还没注册,能开欧洲站吗?
A

能开,但需要公司主体(中国 / 香港公司都行)。先注册公司(5-15 个工作日),再注册 Amazon EU。先选德国作为第一站,搞定德国后顺水推舟到法国意大利。

Q 我想先小批量试卖,合规能不能以后再补?
A

不建议。亚马逊 GPSR 邮件发出后 14 天内不补,listing 直接隐藏;EUDR / EPREL / LUCID 等都有"先有号才能卖"的硬性要求。试卖也要先合规,否则货发过去被退回来,亏损更大。

Q 8 个名词之外,我还会撞到什么?
A

中频出现的还有:CRA(网络弹性法,2027 年生效,电子产品要过网络安全测试)、ESPR(可持续产品法规,能效 / 标签的"上位法")、PLD(产品责任指令修订)、Batteries Regulation(电池法规,2025-2027 分阶段执行)。新手先搞懂 8 个主力,这些先记名字,遇到再查。

Q 我看别的卖家啥都没做,链接还在,是他们不守规矩吗?
A

可能原因很多:他们是"小批量试卖"(还没触发平台审核);商品在 GPSR 豁免清单(如纯服务、二手古董);或者他们也在等限流(只是还没轮到他)。别拿别人当参照,亚马逊的合规审核是分批次的,轮到你时,改已经来不及。

Q 我不想学这么多法规,能不能直接找代办全包?
A

可以,但要谨慎:欧代 / EPR / VAT 代办市场鱼龙混杂,淘宝 / 微信上"€300 包过"的代办,半数是"挂靠地址",平台一旦抽查就封号。建议选行业知名的服务商(ProductIP、Eurofins、Citeo 等官方推荐名单)。
