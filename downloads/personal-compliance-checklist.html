一、你的卖家画像 6 项

勾选所有适用的项目，后续章节会按你的画像过滤必做项。本节是过滤器，不计入 50 项总分。

1.1 产品类目（可多选）

  • 电子电气类 含电池、电源适配器、灯具、家电、音视频、玩具电子部件。需 CE / RED / RoHS / 电池法规。
  • 玩具 / 母婴 / 儿童用品 适用 EN 71 系列、玩具指令 2009/48/EC。0-14 岁产品需特别注意警告语与年龄标识。
  • 美妆 / 护肤 / 个人护理 适用化妆品法规 1223/2009。需 CPNP 通报、产品信息文件 PIF、责任人。
  • 服装 / 纺织 / 鞋类 需 REACH 限用物质合规、纤维成分标签、纺织品标签法规。皮革类注意偶氮染料。
  • 食品 / 膳食补充剂 / 接触材料 需食品法、Novel Food 申报、EC 1935/2004 食品接触材料合规。审查最严格。
  • 工业 / 机械 / 建材 需机械指令 2006/42/EC、CPR 建材法规、CE 符合性评估模块更深。

1.2 销售方式（可多选）

  • FBA 仓发货 Amazon 代存代发。需 FBA 入仓合规文件、危险品审核。VAT 必须注册 5 国。 FBA
  • 跨境直邮（自发物流） 单件 €0-150 适用 IOSS 简化申报；超 €150 走正式清关。需 VAT + EORI。 直邮
  • 海外仓 + 本地配送 第三方海外仓代发。注意仓所在国的 EPR 注册与本地合规义务。
  • B2B 大宗 / 批发 B2B 销售可走反向征税 reverse charge，但需发票链路完整、合同合规。

1.3 销售国家（可多选）

  • 德国 DE 注册门槛最严：LUCID、EAR、VAT、Verpackungsgesetz 全部必需。 DE
  • 法国 FR Citeo 包装回收、DEEE 电子回收、EPR 编号、UFI 毒理中心（化学品类）。 FR
  • 意大利 IT CONAI 包装回收、RAEE 电子、电池 PIB。注意意大利语言警告语强制要求。
  • 西班牙 ES Sigre / Ecoembes 包装、纺织类注意 RETEX。VAT 注册远程销售阈值 €10k。
  • 荷兰 NL WEEELABEX 电子回收、ARN 包装回收。VAT 远程销售阈值 €10k 即将降至 €0。
  • 英国 UK（脱欧后） 需 UKCA 标志、英代 UK Responsible Person、UK VAT、UK EPR（包装/电池/电子）。 UK

1.4 产品物理属性（可多选）

  • 含电池 需 UN 38.3 运输测试、电池法规 2023/1542 注册与回收、电池标签 14 符号。 电池
  • 含包装 需 EPR 包装注册（5 国各自系统）、PPWR 2026-08-12 后减量/可回收要求。 包装
  • 联网产品（IoT / 智能 / 含无线模块） 需 RED 2014/53/EU、CRA 2027-12-11 网络安全合规、固件更新承诺。 联网
画像示例：3C 配件 + FBA + 5 国销售 + 含锂电池 + 联网（蓝牙）
后续重点章节：1（VAT 全注册） + 2（CE + 欧代） + 3（LUCID+Citeo+CONAI…）+ 4（电池全链） + 7（应急）

二、必做合规清单 共 50 项

本章是本清单的核心，共 7 大块、50 项 checkbox。完成度直接决定你的风险等级。建议按顺序逐块打勾，不要跳读。

标签含义： 必做 缺一不可 · 高优 30 天内 · 中优 季度内 · 低优 年度内 · FBA 适用 · 直邮 适用 · 电池 适用 · 联网 适用 · 包装 适用 · UK 适用

2.1 账号与税务（VAT / IOSS / OSS） · 7 项

  • 各国 VAT 增值税号已注册 DE / FR / IT / ES / NL 各自注册。FBA 卖家必做。需通过税务代理或本地财务代表。注册周期 2-8 周。 必做 高优
  • VAT 申报按周期执行（无逾期） DE 每月申报 + 季度汇总；FR 月报；IT 季度；ES 月报或季报；NL 季度或月报。错过一期即风险。 必做 高优
  • IOSS 编号（跨境直邮 < €150） 欧盟外直邮 €0-150 订单必走 IOSS。每月申报。Amazon 平台已自动代收代缴可豁免部分。 直邮 高优
  • OSS（One-Stop-Shop）一站式申报 欧盟内远程销售年销 > €10k 必走 OSS 或在销售国各自注册。超阈值未注册 = 重大风险。 必做 中优
  • EORI 号已申请 经济运营商注册识别号。EU 通用 EORI 用于清关；UK 有独立 GB EORI。无 EORI 无法入仓清关。 必做 高优
  • 远程销售阈值监控 DE 已 €0 阈值，NL 即将降至 €0（待定）。超阈值未在当地注册 VAT 后果严重。 中优
  • Amazon B2B VAT Calculation Service 已启用 Amazon 平台企业对企业自动开票服务。B2B 客户要求合规发票时必启用，否则丢单。 FBA 低优

2.2 产品合规（CE / GPSR / 欧代 / DoC） · 8 项

  • CE 标志正确使用 按比例缩放 ≥ 5mm、不可磨灭、清晰可读。贴在产品本体或包装 / 随附文件。 必做 高优
  • EU Declaration of Conformity（DoC）档案 每 SKU 单独一份。需含产品信息、适用指令、harmonized standards、签署人。保留 10 年。 必做 高优
  • GPSR 欧代（EU Responsible Person）已指定 2024-12-13 强制。非食品类几乎所有消费产品必需。欧代信息需印在产品或包装上。 必做 高优
  • GPSR 产品标签含欧代信息 标签必须含：欧代名称+邮政地址+电子邮箱、制造商信息、原产国（Made in XX）、CE 等。 必做 高优
  • 符合性测试报告（EN 标准） 由公告机构或合规实验室出具。电子需 EN 55032 / EN 55035，玩具需 EN 71，电池需 EN 62133 等。 高优
  • 警告语多语言（5 国） DE / FR / IT / ES / NL 至少。意大利对玩具警告语有强制格式要求。需本地化而非直译。 中优
  • 说明书 + 安全信息 5 国 使用说明、安全警示、年龄限制、清洁保养等。电子电气产品必须含 EU 多语种手册。 中优
  • REACH / RoHS / 限用物质合规 SVHC 通报（>0.1% w/w）、RoHS 10 项、邻苯二甲酸盐、偶氮染料（纺织品）等。需供应链声明。 高优

2.3 包装与环保（LUCID / EPR / PPWR） · 7 项

  • LUCID 注册（DE 包装法） LUCID 号必须印在所有进入德国市场的包装上。注册免费，系统许可后才可分销。 必做 DE 包装 高优
  • FR / IT / ES / NL 各国 EPR 注册 FR Citeo / IT CONAI / ES Ecoembes / NL ARN。各国注册系统独立，需单独注册 + 申报。 必做 包装 高优
  • 法国 Citeo / 其他回收组织申报 按销售包装重量向 EPR 组织缴费。未申报 = 法国海关 / 平台双重拦截。 FR 中优
  • 德国 EAR（电子电气 EPR） Stiftung EAR。WEEE 指令覆盖产品必注册。已与 LUCID 联动验证。 DE 高优
  • PPWR 减量化准备（2026-08-12 生效） 最小包装、空隙率 ≤ 50%、免运费包装禁外。需重新设计包装并提供报告。 包装 中优
  • 可回收材料（PPWR 2026-08-12） 2030 年前所有包装可回收 ≥ 70%。2030 后部分品类需最少回收材料含量比例。 包装 低优
  • 重复使用 / 再填充包装（PPWR 2030） 运输包装、外卖等场景最终需 ≥ 10% 重复使用率。提前规划可节省长期改造成本。 包装 低优

2.4 电池与电子（CRA / Batteries / RED） · 7 项

  • 电池法规 2023/1542 注册与生产者责任 2023-08-17 生效。所有含电池产品（含纽扣电池）需在销售国注册生产者延伸责任。 必做 电池 高优
  • 碳足迹声明（LMT） 工业电池 / 电动车电池 / LMT 电池（>2kWh）2025 起需碳足迹声明 + 标签。 电池 中优
  • 废旧电池回收标签 产品上必须含「叉轮垃圾桶」符号，可附电池容量标识。纽扣电池警告语强制要求。 电池 高优
  • DoC 含电池指令符合性 含电池产品的 DoC 需引述 Battery Regulation 2023/1542 作为适用法规之一。 电池 中优
  • UN 38.3 电池安全运输测试 锂电产品航空 / 海运 / 陆运均需 UN 38.3 测试报告。FBA 入仓审核必查。 电池 必做 高优
  • RED 指令合规（含无线模块） 2014/53/EU。WiFi / 蓝牙 / 4G / NFC / 433MHz 等无线产品必做。需 RED DoC + 公告机构证书（部分）。 联网 必做 高优
  • CRA 网络弹性法案（2027-12-11 生效） 所有联网产品需：默认安全、5 年补丁支持、SBOM 软件物料清单、安全披露流程。 联网 中优

2.5 平台运营（广告词 / 评论 / 警告信） · 7 项

  • Listing 文案不含违禁词 禁：医疗 / 疗效类（治、疗、消炎、止痛）、最佳 / 第一 / 唯一 等最高级词、绝对化承诺。 必做 高优
  • 关键词不含他人品牌 / 注册商标 使用他人品牌词做后端关键词属侵权。Apple / Samsung / Nike 等大牌零容忍。 必做 高优
  • 评论合规（无诱导好评） 禁：返现索评、夹带小卡片要求 5 星、第三方测评机构批量刷评。一经发现即关店。 必做 高优
  • A-to-Z 索赔率 & 警告信监控 ODR（Order Defect Rate）必须 < 1%。超阈值账户直接 health warning。 中优
  • 知识产权合规（商标 / 外观 / 专利） 上架前查 EUIPO 商标数据库 + Amazon Brand Registry。外观 / 实用新型同样需注意。 高优
  • 平台政策更新订阅 Amazon Seller News + EU Compliance 邮件订阅。新规常 30-90 天内执行，无通知期。 低优
  • 账户健康度仪表盘周度查看 登录 Seller Central → Account Health。关注 4 大指标：ODR、Late Shipment、Policy Violations、Customer Service Dissatisfaction。 中优

2.6 物流与文件（头程 / EORI / 报关） · 7 项

  • 头程物流商具备 EU 清关能力 确认货代有 EU IOSS 接入、5 国 VAT 资源、危险品运输资质。FBA 头程必须正报清关。 FBA 高优
  • 报关 HS Code 正确（6 位 + EU 8 位） TARIC 8 位必填。HS Code 错误 = 关税错缴、海关查验、产品被扣。 必做 中优
  • 商业发票 + 装箱单 5 国语言版 发票含 HS Code、原产国、价值、币种。装箱单含毛重 / 净重 / 体积 / 件数。 必做 高优
  • 物流商代缴关税凭证保留 VAT 申报时需提供进口 VAT 抵扣凭证（IVA a la importación）。建议电子化归档 ≥ 7 年。 中优
  • FBA 长期仓储费监控 > 365 天库存 $6.90/cu.ft/季，> 180 天分档收费。每月查看 Inventory Age。 FBA 低优
  • 商品原产地（Made in XX）标注 欧盟原产地标识强制。需在产品本体 / 包装 / 随附文件上明示。错误 = 海关扣货 + 罚款。 必做 高优
  • 临时进口 / ATA 单证册（展销 / 测试） 仅展销 / 测试 / 维修用途。需向当地商会申请。商业零售禁止使用。 低优

2.7 应急（48h 通知 / 召回 / 投诉） · 7 项

  • 48h 危险产品安全通知流程（GPSR） 严重风险产品需 48h 内通报 Safety Business Gateway。各成员国市场监督机构 + 欧代同步。 必做 高优
  • 产品召回计划与模板 含：召回启动条件、消费者通知渠道、退货 / 退款流程、向平台 / 监管同步机制。预设模板不慌乱。 必做 高优
  • 客户投诉记录（含沟通截图） 3 年内可追溯。需含订单号、SKU、问题描述、沟通全程。监管检查时必查。 中优
  • 欧代紧急联系方式（英语 24h） 欧代必须能英语 24h 响应市场监督机构的查询。应急电话 + 邮箱预先登记。 必做 高优
  • 平台账户 suspend 应对流程 POA（Plan of Action）模板预先备好、备用账户备案、应急律师联系方式。 高优
  • 资金冻结预备金（3-6 月现金流） Amazon 储备金可冻结最长 90 天，平台审核 / 物流商压款叠加。建议预备 3-6 个月运营现金。 中优
  • 法律顾问 / 合规顾问备案 专业 EU 合规律师 + 当地会计。提前建立联系，紧急时不再「抓瞎」寻找。 低优
