8 大名词先搞懂

亚马逊欧洲站新手必读, 5 分钟搞清楚这些英文缩写都在说啥。

🌱 新手看这里就够, 每个名词 1 句话 讲清楚"它跟你啥关系"

⚡ 直接搜索你要查的缩写 (如 "GPSR" / "电池") — 客户端实时过滤

🔍 显示 8 / 8 个
GPSR — 通用产品安全法规
#1

欧盟 2024-12-13 起强制, 亚马逊欧洲站卖家必须显示"欧盟责任人"信息。

🛒 它跟你啥关系
你在亚马逊欧洲站卖任何非食品类商品, 都要在 listing 上写"欧盟责任人"的姓名 + 联系方式. 没写就下架.
📅 什么时候管
2024-12-13 已生效
💸 罚款多少
最高 1000 万欧 或 5% 全球营收
看完整版 →
CE — 合格证标识
#2

产品上必须有这个标志, 证明它"在欧洲可以卖"。

🛒 它跟你啥关系
几乎所有非食品类商品 (电子/玩具/机械/医疗器械) 都要打. 没打, 亚马逊会下架, 海关会扣货.
📅 什么时候管
一直强制, 不是新规
💸 罚款多少
最高 50 万欧
看完整版 →
EPR — 生产者延伸责任
#3

你卖的产品包装用完了, 你要负责回收。

🛒 它跟你啥关系
在德国 / 法国 / 西班牙卖任何带包装的商品, 都要注册 EPR 编号, 亚马逊强制要求. 不注册就下架.
📅 什么时候管
一直在执行
💸 罚款多少
各国不同, 最高 20 万欧 (德国)
看完整版 →
VAT — 增值税
#4

欧洲的"消费税", 卖东西要交的税。

🛒 它跟你啥关系
你是中国卖家, 卖到欧洲, 货值 ≤ €150 要注册 IOSS, > €150 走标准 VAT. 没注册 IOSS, 客户清关时被收两次税.
📅 什么时候管
一直强制
💸 罚款多少
各国不同, 补税 + 滞纳金
看完整版 →
EPREL — 能效标签数据库
#5

卖电气电子产品要在欧盟的数据库登记"能效信息"。

🛒 它跟你啥关系
卖冰箱 / 洗衣机 / 电视 / 显示器 / 灯 / 吸尘器等电气电子, 都要在 EPREL 注册产品能效数据, 亚马逊强制.
📅 什么时候管
2019 起分阶段
💸 罚款多少
各国不同
看完整版 →
IOSS — 进口一站式申报
#6

小包裹 (≤€150) 的 VAT 申报服务。

🛒 它跟你啥关系
你从中国发小包到欧盟消费者, 注册 IOSS 后, 客户结账时已含 VAT, 清关无障碍. 没注册, 客户清关时被收税 + 处理费.
📅 什么时候管
2021-07-01 已生效
💸 罚款多少
没注册 → 不能享受简化申报
看完整版 →
PPWR — 包装与包装废弃物法规
#7

2026-08-12 起所有欧盟市场包装要满足"最小化 + 可回收 + 标签"新要求。

🛒 它跟你啥关系
2026-08-12 后, 你的产品包装要: ① 物质限制 (不能含某些重金属) ② 最小化 ③ 标签 (可回收标志) ④ 可回收/再生原料. 亚马逊 / TEMU / 速卖通都已通知.
📅 什么时候管
2026-08-12 全面实施
💸 罚款多少
各国不同
看完整版 →
EUDR — 反毁林法规
#8

你卖的商品如果涉及"森林", 要证明它"不是来自毁林地区"。

🛒 它跟你啥关系
卖牛肉 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木及衍生品 (e.g. 木家具, 纸, 橡胶玩具), 要上传"尽职调查声明 (DDS) + 地块地理坐标". 亚马逊已通知.
📅 什么时候管
2025-12-30 大企业全面适用
💸 罚款多少
最高 4% 全球营收
