关于我们

网站使命

欧规雷达(EuroRule Radar)面向中国亚马逊欧洲站中小卖家,把分散、复杂、更新快的规则变化整理为中文、清晰、可执行、可追溯来源的规则卡片与检查清单,帮助你在第一时间判断"是否影响我、什么时候、做什么"。

我们不做泛资讯门户,也不做通用 AI 聊天机器人;我们聚焦一件事:把规则变化转成你今天能完成的行动清单。

信息来源与编辑原则

来源白名单

  • Amazon Seller Central(欧洲各国)
  • 欧盟委员会 + EUR-Lex(法规原文)
  • 欧盟各成员国主管机构(如德国 LUCID、法国 Citeo 等)
  • 欧盟官方税务 / 环保 / 产品安全机构
  • 权威标准或认证机构的原文说明

编辑原则

  1. 每条规则附原始来源 + 核对日期
  2. 未核实的标注"示例 / 待核实",不写成确定事实
  3. 不杜撰法规结论、链接、日期或处罚后果
  4. 优先中文化 + 拆分复杂句子,避免堆砌术语
  5. 涉及金额、罚款、日期等数字,标注"以原始来源为准"

更新机制

本站按"每周一次"节奏更新;每条规则有"核对日期"字段,运营对超过 90 天未核对的规则优先复核。

遇到重大政策变化(例如新 EU 指令生效、平台重大公告),会在 24-48 小时内发布专题卡。

内容运营 SOP(白名单、核对步骤、写作风格)请查看项目仓库中的 CONTENT_GUIDE.md / README.md(开发者文档,不在本站展示)。

提交规则线索

如果你是卖家,看到一条新的官方公告 / 法规变化,想让团队优先处理,可以发送线索给我们:

  • 📮 邮箱占位:YOUR_CONTACT_EMAIL@example.com(部署前在 src/lib/radar.ts 替换为你的真实邮箱)
  • 🛠 GitHub Issues(占位):github.com/YOUR_GITHUB_USER/eurorule-radar/issues

提交时请提供:

  1. 原始链接(官方公告、EUR-Lex 条目、平台 Seller Central 页面)
  2. 发布日期和生效日期
  3. 你认为涉及的卖家类型 / 类目 / 国家
  4. (可选)你认为最重要的 1-2 个行动项

我们收到后会在 7 天内评估并发布。
