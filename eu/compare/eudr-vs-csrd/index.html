EUDR vs CSRD

EUDR 管"7 类商品不能含毁林来源",CSRD 管"大公司要按 ESRS 写可持续报告" —— 一个管"商品",一个管"公司"。

EUDR

EUDR 2023/1115 反毁林法规

商品尽调 + 地理坐标 + 7 类商品禁毁林,2025-12 大企业强制
VS
CSRD

CSRD 2022/2464 企业可持续报告指令

大公司按 ESRS 写可持续报告 + 第三方审计 + XHTML 数字标签,FY2024 首批

速查表 · 7 个关键差异

各自是什么

📦 EUDR(EUDR 2023/1115 反毁林法规)

EUDR 2023/1115《反毁林法规》,2023-06 通过,2023-06-09 发布于欧盟官方公报,2025-12-30 大企业强制、2026-06-30 中小企业强制,核心目的是减少欧盟消费市场对全球毁林的贡献。 适用范围:7 类商品 —— 牛、可可、咖啡、棕榈油、橡胶、大豆、木材,以及它们的衍生品(巧克力、皮革、橡胶轮胎、家具、印刷品等)。不论是欧盟内首次投放市场还是出口,只要涉及这 7 类商品就触发。 关键义务:经营者(欧盟内首次投放商品)要做尽职调查 —— 收集商品来源地的地理坐标(GPS 多边形)、生产国别、毁林时间点(2020-12-31 之后是否毁林);贸易商(在欧盟市场销售已投放商品)要保留尽职调查声明至少 5 年。所有声明要上传到欧盟公开信息系统,商品被认定为低风险才能投放。 生产国别分类(2025 年欧委会发布):低风险(德国、波兰等 11 国)、标准风险(巴西、阿根廷、印尼等 11 国)、高风险(俄罗斯、白俄、朝鲜、缅甸等 16 国)。低风险简化尽职调查,高风险加重审查,部分高风险国家有特殊供应链合规要求。 不做 EUDR 的实质后果:商品到欧盟港口被海关扣押 / 销毁、欧盟全境禁售 7 类商品、经营者列入黑名单并排除在公共采购之外,罚款按公司全球年营业额的 1-4%,一次违规能清掉中型贸易商一两年利润。

🔁 CSRD(CSRD 2022/2464 企业可持续报告指令)

CSRD 2022/2464《企业可持续报告指令》,2022-12 通过,2023-01 生效,核心目的是让欧盟大公司按统一标准披露可持续信息,让投资者 / 消费者 / 监管者能看到公司真实的 ESG 表现。 适用公司(分四批,2024-2029):FY2024 起,大型公司(>250 员工 + 任两条:营业额 >5000 万 € / 资产 >2500 万 €);FY2025 起,上市公司中型公司;FY2026 起,上市小型公司(可申请 2 年豁免);FY2029 起,非欧盟公司在欧盟营收 >1.5 亿 €。 关键要求:必须按 ESRS(欧洲可持续报告准则)写报告,分为跨行业准则(ESRS 1 一般要求 + 双重重要性评估、ESRS 2 一般披露:治理 / 战略 / 风险与机遇管理 / 指标与目标)和主题准则(ESRS E1-E5 气候 / 污染 / 水 / 生物多样性 / 资源循环、ESRS S1-S4 员工 / 价值链工人 / 社区 / 消费者、ESRS G1 商业行为),共 10 个主题;公司先做双重重要性评估,识别出"对公司和环境 / 社会有重大影响"的议题,再针对性披露。 必须经第三方独立审计(有限保证起步,合理保证逐步推进);报告必须用 XHTML 数字标签(iXBRL),发布到公司官网 + 各国国家公告机构。 不披露的实质后果:融资受限(银行 / 投资人把 ESRS 报告作为放贷和信用评级参考)、供应链被大客户反向要求提供 CSRD 报告(小公司连带被卷进去)、欧盟数字标签公开披露后 ESG 表现差的公司股价 / 信用评级同步下降。

👉 看完整 欧盟 ESPR 生态设计法规 已生效:数字产品护照 (DPP) 分阶段落地,几乎所有物理产品受波及

你是哪种卖家?选哪个?

选 EUDR

卖咖啡豆的欧盟 Amazon 卖家(中小规模,欧盟注册公司)

你卖的就是 EUDR 7 类商品里排第三的咖啡(生豆 / 烘焙豆 / 咖啡粉都算),EUDR 强制触发 —— 2025-12-30 起大企业、2026-06-30 起中小企业,你作为经营者必须做尽职调查 + 收集产地 GPS 坐标 + 把声明上传到欧盟信息系统。CSRD 跟你没直接关系 —— 这是公司层面的报告义务,你公司规模够不到 CSRD 触发线(>250 员工 + 任两条财务门槛)就不用做;但大客户(走欧盟连锁超市)可能反向要求你提供 ESG 披露,留意一下合同条款。

选 CSRD

德国大型贸易公司(>250 员工,年营收 5000 万 €+,上市公司)

你公司满足 CSRD 三大触发线(>250 员工 + 营业额 >5000 万 € + 资产 >2500 万 €),2024 财年起就要按 ESRS 准则写报告 + 第三方审计 + 公开披露。CSRD 是管"公司"的,跟卖什么商品没直接关系。EUDR 跟你的关系取决于商品 —— 如果你也做 7 类原料(咖啡 / 可可 / 棕榈油等)的贸易,那 EUDR 也叠加触发;纯做电器 / 机械 / 纺织的贸易,EUDR 不管你,只做 CSRD。

都不做 / 暂不触发

中国小卖家(独立站,卖玩具,自己单干,小批量跨境直邮)

你卖玩具 —— EUDR 7 类商品是牛 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木材,玩具不在其中(除非你用木头做玩具,那种会部分触发),EUDR 不管你;你公司规模也远不到 CSRD 触发线(>250 员工 + 高营业额),CSRD 也不管你。你需要关注的是 GPSR(通用产品安全 + 欧代)+ CE 玩具 EN 71 标准 + EPR(包装回收),不是这两个法规。

两个都做

大型食品集团(同时经营咖啡 / 可可 / 棕榈油三条产品线,集团上市)

你两个都触发,而且都很重。EUDR 角度:咖啡 / 可可 / 棕榈油都是 7 类商品 —— 你要建全链路追溯系统、收集种植园 GPS 坐标、把三类的尽调声明都上传到欧盟信息系统,2025-12-30 起强制。CSRD 角度:大公司 + 上市公司,2024 财年起按 ESRS 写全公司可持续报告 + 第三方审计 + 公开披露。两套体系平行,各做各的:EUDR 是"原料 / 商品"层面的合规,CSRD 是"公司"层面的披露;技术上可以数据共享(比如 EUDR 收集的毁林风险数据可以纳入 CSRD 报告的生物多样性章节),但交付物是两套独立文档。

关键时间线

  1. 2022-12 CSRD(2022/2464)正式通过,2022-12-16 发布于欧盟官方公报,要求各成员国在 2024-07-06 前转化为本国法律
  2. 2023-01 CSRD 部分条款生效(2023-01-05 entry into force),企业开始准备 ESRS 准则适配
  3. 2023-06 EUDR(2023/1115)正式通过,2023-06-09 发布于欧盟官方公报,原定 2024-12-30 起对大企业强制适用
  4. 2024-12 欧委会决定 EUDR 推迟 12 个月 —— 大企业 / 经营者延至 2025-12-30,中小企业 / 贸易商延至 2026-06-30
  5. 2024-2029 CSRD 分四批适用:FY2024 首批大公司 + FY2025 上市中型 + FY2026 上市小型 + FY2029 非欧盟公司
  6. 2025-12-30 EUDR 对大企业 / 经营者强制适用,未做尽调的 7 类商品不得进入欧盟市场
  7. 2026-06-30 EUDR 对中小企业 / 贸易商强制适用,所有 7 类商品经营者 / 贸易商全覆盖

不做会怎样?罚款与风险

EUDR

EUDR 违规罚款按公司全球年营业额的 1-4%(欧盟最高上限),具体由各成员国立法。叠加处罚:扣押 / 没收商品、禁售 7 类商品(在欧盟全境)、把经营者列入黑名单 + 排除在公共采购之外,严重情形下还要承担刑事责任(部分成员国)。罚款 + 商品没收 + 品牌信誉损失,对中型贸易商来说一次违规能清掉一两年利润。

CSRD

CSRD 违规罚款因国家而异,欧盟只规定最低框架:德国 1000-1000 万 €、法国可判 5 年监禁 + 罚款、荷兰最高 10 万 €、意大利最高 5 万 €。实质性后果不是罚款,而是:① 银行 / 投资人把 ESRS 报告作为放贷和信用评级参考,没有报告融资直接受限;② 大客户(CSRD 适用公司)反向要求供应链上的小公司也按 ESRS 披露,小公司被动被卷进去;③ 欧盟数字标签公开披露,ESG 表现差的公司股价 / 信用评级同步下降,管理责任还会落到董事个人头上。

常见问题

Q EUDR 和 CSRD 的范围差异?一个是管商品还是管公司?
A

两个法规的"管辖对象"完全不同。EUDR 管的是"商品本身",只要你的商品是 7 类之一(牛 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木材)或其衍生品,就要做尽职调查;CSRD 管的是"公司本身",看的是公司规模(>250 员工 + 任两条财务门槛),跟卖什么商品没直接关系 —— 卖玩具的大公司触发 CSRD 但不触发 EUDR,卖咖啡豆的小公司触发 EUDR 但不触发 CSRD。一个看商品,一个看公司,不要混。

Q EUDR 和 CSRD 的触发门槛分别是什么?
A

EUDR:商品类别是牛 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木材,任一即触发,不分公司大小,大中小企业全覆盖。CSRD:公司规模是 >250 员工 + 任两条财务门槛(营业额 >5000 万 € / 资产 >2500 万 €),分四批适用(FY2024 大公司 / FY2025 上市中型 / FY2026 上市小型 / FY2029 非欧盟公司在欧盟营收 >1.5 亿 €)。一个看商品类别,一个看公司规模,完全两套逻辑。

Q EUDR 监管的 7 类商品完整名单是什么?
A

EUDR 监管的 7 类商品:牛(cattle,含活牛、牛肉、皮革)、可可(cocoa,含可可豆、可可膏、巧克力、巧克力制品)、咖啡(coffee,含生豆、烘焙豆、咖啡粉、含咖啡提取物的产品)、棕榈油(palm oil,含原棕榈油、棕榈仁油、棕榈油衍生品)、橡胶(rubber,含天然橡胶、合成橡胶、橡胶制品如轮胎)、大豆(soya,含大豆、豆粕、豆油、酱油)、木材(wood,含原木、锯材、家具、纸浆、印刷品)。每一类都包含其衍生品,比如木头做的家具 / 地板 / 印刷品都算,巧克力里的可可豆也触发 EUDR。

Q CSRD 的 ESRS 报告模板长什么样?
A

ESRS(欧洲可持续报告准则是 CSRD 的核心。结构上分三层:① ESRS 1(一般要求 + 双重重要性评估 —— 评估"对公司的财务影响"和"对公司外部环境 / 社会的影响")② ESRS 2(一般披露:治理、战略、影响 / 风险 / 机遇管理、指标与目标)③ 主题准则(ESRS E1-E5 气候 / 污染 / 水 / 生物多样性 / 资源循环;ESRS S1-S4 员工 / 价值链工人 / 社区 / 消费者;ESRS G1 商业行为),共 10 个主题。公司先做双重重要性评估,识别出"对公司和环境 / 社会有重大影响"的议题,再针对性披露。官方模板在 EFRAG(欧洲财务报告咨询组)网站可下,2023 年底发布的 12 套准则 + 2024 年发布的行业准则。

Q 我们公司同时适用 EUDR 和 CSRD,两套体系怎么配合?
A

很多公司会同时适用 EUDR 和 CSRD,典型例子:大型食品集团卖咖啡 / 可可 / 棕榈油。两套体系平行:① EUDR 是"商品层面" —— 建立 7 类原料的追溯系统、收集种植园 GPS 坐标、欧盟信息系统上传尽职调查声明。② CSRD 是"公司层面" —— 按 ESRS 写一份涵盖气候变化 / 社会 / 治理的全公司报告。技术上可以数据共享(比如 EUDR 收集的毁林风险数据可以纳入 CSRD 报告的生物多样性章节),但交付物是两套独立文档:一份是给欧盟信息系统的尽职调查声明,一份是给公司官网 + 国家公告机构的可持续报告。两者不互相替代,缺一个就缺一个的合规。

Q 我是小卖家,EUDR 和 CSRD 我需要做哪个?
A

看你卖什么 + 公司多大规模。EUDR:小卖家(中小企业)只要卖 7 类商品(牛 / 可可 / 咖啡 / 棕榈油 / 橡胶 / 大豆 / 木材)就触发 —— 2026-06-30 起强制,需要找上游供应商要产地 GPS 坐标 + 毁林时间点 + 准备欧盟标准模板的尽调声明 + 上传到欧盟信息系统。CSRD:小卖家基本不触发 —— 除非你的公司真的达到 >250 员工 + 高营业额 + 高资产,且是上市公司或在欧盟有重大营收;没达到就是跟 CSRD 无关。玩具 / 服装 / 3C / 家居等小卖家,EUDR 不会触发(除非你用木头 / 橡胶 / 大豆等做原料),CSRD 几乎不会触发。

欧盟 REACH 法规 (EC) 1907/2006 是覆盖所有在欧盟制造或进口产品(物质、配制品、物品)的化学品管理框架,2024-2025 年 SVHC 候选清单已超 240 项,Article 7 通报义务要求产品中 SVHC ≥ 0.1% (w/w) 时向 ECHA 通报,Annex XVII 限制清单含数百种禁用 / 限用物质。亚马逊要求全品类(玩具 / 服装 / 电子 / 家具 / 珠宝)卖家履行 REACH 下游用户义务,违规扣货。
