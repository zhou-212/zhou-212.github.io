CE vs FCC

CE 是欧盟非食品消费品的「合格护照」,FCC 是美国电子电气设备的「无线电频证」 —— 双地区卖货两个都要,完全不同的体系。

CE

CE 标志 (欧盟合格标志)

欧盟非食品消费品(电子/机械/玩具/医疗等)市场准入标志,挂欧代 + DoC + 技术文件
VS
FCC

FCC 认证 (美国联邦通信委员会)

美国电子电气设备射频/EMC 强制认证,无线产品需 FCC ID,无意辐射可自我声明

各自是什么

📦 CE(CE 标志 (欧盟合格标志))

CE 标志是欧盟非食品消费品的强制合格标志,1985 年随《新方法指令》引入,目前覆盖 27 个成员国 + 冰岛/挪威/列支敦士登(EEA)。所有非食品实物——电气、机械、玩具、压力设备、医疗器械、无线电、建材、个人防护——卖到欧盟基本都要贴 CE。 要做 CE 关键四步:一是按适用的协调标准(EN 6xxxx 系列,如 EN 60335 家电安全、EN 55032 EMC、EN 62368-1 音视频)做测试;二是出 EU DoC(符合性声明),你自己签字声明产品符合哪些指令;三是技术文件(Technical Documentation)保留 10 年,包含电路图、关键件清单、BOM、风险评估、测试报告;四是挂欧代(EU Responsible Person,GPSR 强制),2024-12-13 后必须有欧盟当地的自然人或法人做你的责任人,出问题能找到人。 CE 没有统一的"发证机构",大部分产品是自我声明 + DoC 模式,只有高风险类目(医疗器械、电梯、压力设备等)才需要 Notified Body 出具证书。Notified Body 名单在欧盟 NANDO 数据库公开,可查编号(如 TÜV SÜD 0123、SGS 0120)。

🔁 FCC(FCC 认证 (美国联邦通信委员会))

FCC 认证是美国联邦通信委员会(Federal Communications Commission)的设备授权体系,1934 年《通信法》设立,主要管美国境内的射频发射和电磁兼容。FCC 法规主要分 Part 15(无意辐射装置 + 低功率发射器,基本覆盖所有电子电气)、Part 18(工业/科学/医疗设备 ISM)、Part 22/24/27(蜂窝通信)等。 FCC 把产品分两类:有意辐射(蓝牙、Wi-Fi、蜂窝、Zigbee 等无线产品)要拿 FCC ID,在 FCC 认可的实验室(MTC)测完后由 Telecommunication Certification Body(TCB,如 UL、TÜV、Intertek)发证,Grantee Code 公开可查;无意辐射(普通家电、电脑外设、LED 灯)走 Supplier's Declaration of Conformity(SDoC),2020 年起替代旧的 Verification 模式,自己测 + 自己声明 + 贴 FCC 标志即可。 关键三件事:一是产品本体或包装贴 FCC 标志(Grantee Code 编码也在内);二是用户手册里要有 FCC 合规声明(类似" This device complies with Part 15 of the FCC Rules"那段话);三是产品参数和测试报告保留备查。FCC Equipment Authorization Search 数据库(apps.fcc.gov/oetcf/eas/)公开所有授权记录,任何人都能查。

你是哪种卖家?选哪个?

都不做 / 暂不触发

卖纯纺织品的卖家(纯棉 T 恤、亚麻床品、羊毛围巾,无电子元件)

纯纺织(无电池、无电子、无无线模块)既不在 CE 的强制电子/机械类目下需要 RED 评估(注意 GPSR 欧代仍要做),也不在 FCC 的射频管制范围(没射频发射就不归 FCC 管)。这类产品只触发 GPSR 欧代 + 标签要求,跟 CE 和 FCC 都没关系。

两个都做

卖无线充电宝/蓝牙耳机的双地区卖家(亚马逊 EU 5 国 + Amazon.com)

无线产品两个地区都强制。欧盟走 CE RED 2014/53/EU,需要 Notified Body 评估 + DoC + 欧代 + EN 62368-1/EN 300 328 等测试;美国走 FCC ID 路径,要 Grantee Code + 实验室测试 + TCB 发证。两套报告基本不能互认,要分别做,合规预算准备 8000-20000 美元。

选 CE

中国小卖家只做亚马逊 EU 5 国,纯有线充电头/有线耳机(无蓝牙)

你只做欧盟、只卖有线产品,CE 必做:EMC 指令(EN 55032/55035) + LVD 低电压指令(EN 62368-1) + DoC + 欧代。FCC 跟你无关——产品不在美国卖,不用管。预算重点放在 CE 测试(5000-15000 欧元)+ 欧代(500-3000 €/年)即可。

选 FCC

美国亚马逊 FBA 卖家,只做 Amazon.com,卖无线路由器/智能插座/宠物 GPS 追踪器

你只做美国,无线产品必做 FCC ID:Grantee Code 申请(免费,一次性)+ FCC 认可实验室测试(单产品 3000-15000 USD)+ TCB 发证(几百到几千 USD)。CE 和 UKCA 都不用做,欧代也不用挂。注意 Amazon.com 2018 起强制 FCC ID 上传,无线产品 listing 必须有授权号才能上架。

关键时间线

  1. 1985-01 CE 标志随欧盟《新方法指令》(New Approach Directives)正式引入,首次在玩具、低电压、机械等产品类别强制实施
  2. 1996 FCC 改组,Part 15 持续更新,确立无意辐射装置 + 低功率发射器的现代分类框架,数字设备(Class A/B)开始强制要求
  3. 2014-04 欧盟通过无线设备指令 RED 2014/53/EU,准备替代老的 R&TTE 1999/5/EC,覆盖所有无线收发设备
  4. 2016-06-13 RED 2014/53/EU 全面强制实施,R&TTE 退出历史舞台,所有无线产品投放欧盟市场必须按新指令评估
  5. 2017-2020 FCC 多个 5G 频段规则陆续更新:CBRS(3.5 GHz)、C-Band(3.7-3.98 GHz)、6 GHz(Wi-Fi 6E)开放,Part 15 多次修订
  6. 2021-2024 英国 Brexit 后,UKCA 标志替代英国 CE;2024-12-31 过渡期结束,UKCA 在英国全面替代 CE,英美欧三套体系完全独立

不做会怎样?罚款与风险

CE

欧盟 CE 违规罚款 1-1000 万 €(看产品类目和成员国),listing 在 Amazon EU 5 国直接下架,海关在 EU 港口抽查时发现无 CE 或 DoC 不合规直接扣押销毁;产品召回时找不到欧代,品牌方承担全部召回成本,严重者上欧盟 Safety Gate 名单,整个欧盟清关难度飙升。

FCC

美国 FCC 违规每个产品 10000+ USD 罚款,CBP(海关边境保护局)直接扣货 + 进口禁令,FCC 还会发布 Consent Decree(同意令)要求整改,严重者刑事起诉 + 设备没收 + Grantee Code 吊销,以后整个公司都做不了无线产品。

常见问题

Q CE 和 FCC 的产品范围差异在哪里?
A

CE 几乎覆盖所有非食品实物消费品(电气、机械、玩具、医疗、压力、建材等),是「广义产品安全」标志;FCC 只管射频发射和电磁兼容,只针对电子电气设备。一句话:CE 看你产品安不安全,FCC 看你产品会不会干扰别人。无电池无电子的纯纺织/纯木制品不归 FCC 管,但可能要做 CE 协调标准(机械类);无线蓝牙耳机两个都触发。

Q 无线产品(蓝牙/Wi-Fi/蜂窝)CE 和 FCC 怎么选?
A

两个都做。欧盟走 CE RED 2014/53/EU,要做 EN 300 328(蓝牙/Wi-Fi)、EN 301 489(EMC)、EN 62368-1(安全),需要 Notified Body 评估 + DoC + 欧代;美国走 FCC Part 15 + FCC ID,Grantee Code + MTC 测试 + TCB 发证。两套测试报告基本不能互认,标准不一样(欧盟用 EN 协调标准,美国用 FCC 规则),分别找不同的测试实验室做。

Q CE 报告和 FCC 报告能互认吗?
A

大部分情况不能。CE 用 EN 协调标准,FCC 用 CFR 47 Part 15/18 等,标准体系独立;测试实验室也不通用(欧盟 Notified Body vs 美国 MTC + TCB)。少数基础测试项目(EMC 射频部分)可以参考对方的报告数据,但合规性结论要重新评估。跨境卖家最省钱的做法是:同一家大实验室(如 TÜV、Intertek、SGS、UL)同时做两边测试,实验室内部数据共享能省 20-40% 时间和成本。

Q 我是小卖家,只做单地区,先做 CE 还是先做 FCC?
A

看你的目标市场:卖欧盟先做 CE,卖美国先做 FCC;只做其中一个地区只做那个标志就行。CE 必做的四件套:EMC 指令 + LVD 低电压指令 + DoC + 欧代,常见产品 5000-15000 欧元;FCC 必做:申请 Grantee Code(免费)+ 测试(单产品 3000-15000 USD)+ TCB 发证。注意:无线产品(FCC ID 类)必须先做,无线路由器、蓝牙设备、智能音箱在 Amazon.com 上没 FCC ID 上不了架。

Q 亚马逊后台要双标上传吗?
A

看你在哪个站点卖。Amazon EU 5 国(DE/FR/IT/ES/NL)后台要求 CE 标志 + DoC + 欧代信息;Amazon.com 要求 FCC 标志 + FCC ID(无线产品)或 SDoC 自我声明(无意辐射)。如果两个地区都卖,后台两边都要上传对应的合规文件,亚马逊不会替你做转换。注意 2024 年亚马逊对 CE 标志管控更严,DoC 没上传或欧代信息不全直接限流。
