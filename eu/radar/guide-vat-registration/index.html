一句话结论

VAT(Value Added Tax,增值税)是欧盟的”销售税”——你在欧盟每卖一单,要把售价的 19-22%(看国家)作为税交给该国税务局。几乎所有 B2C 卖家都要注册 VAT 号(不注册就销售 = 偷税,罚款 + 账号封停)。新手推荐找当地税务代理代办(1000-2000 欧 / 国家,4 周拿号),自己远程注册只适合英国等少数允许的国家。别贪便宜找野代办——VAT 号无效 = listing 全下架。

这是啥

VAT 是什么:欧盟成员国对商品和服务征收的消费税,本质上和中国的”增值税”、美国的”Sales Tax”是一回事。

VAT 的核心逻辑:

  • 你在德国卖 100 欧的商品 → 收买家 119 欧(含 19% 增值税)
  • 这 19 欧 = 你的”代收税款”,不是你的收入
  • 每月 / 每季度申报一次,全额上交给德国税务局
  • 你进货时付的 VAT(比如进货 50 欧含 8 欧 VAT)→ 可以抵减你销售时收的 VAT
  • 实际交给税务局 = 销售 VAT - 进货 VAT(差额交)

谁需要注册 VAT:

  • B2C 卖家(卖给个人买家)→ 基本都要(亚马逊、eBay、速卖通欧洲站都强制)
  • B2B 卖家(卖给企业)→ 部分情况要,欧盟内 B2B 可以走反向征税(Reverse Charge),但跨境 B2B 从中国发货还是要注册
  • 自发货 / FBA 都算——只要库存放在欧盟境内,就要注册
  • 唯一例外:用亚马逊 pan-EU 库存调拨 + OSS 跨境小包有不同规则,见下面 OSS 章节

为什么 VAT 这事重要:

  • 不注册就销售 = 偷税——亚马逊会直接从你的销售款里代扣(Marketplace Facilitator 规则,2021 年起在欧盟多国生效)
  • 亚马逊代扣的税率通常比你自己注册高(因为亚马逊不知道你的成本结构,按全额扣)
  • 自己注册 + 自己申报 = 合法 + 还能抵减进货 VAT,比被代扣省钱 1-3%

为什么跟你有关

不注册 VAT,4 个直接后果:

  1. 亚马逊直接代扣,你的利润被吃掉 3-5%:亚马逊”Marketplace Facilitator”在德国、法国、意大利、西班牙、波兰已经强制实施——没 VAT 号的卖家,每笔销售直接被亚马逊扣 19-22%,你自己不能再抵减进货 VAT
  2. listing 在海关清关时被扣:FBA 头程发货到欧盟,海关要 VAT 号清关;没 VAT = 整批货被扣 1-4 周,仓储费 + 滞港费一天 200-1000 欧
  3. 税务稽查罚款 + 账号封停:欧盟各国税务局(德国 FAZIT、法国 DGFIP、意大利 AdE)对未注册 + 销售的卖家,单次罚款 1 万 - 5 万欧元,情节严重的(销售额 > 10 万欧未报税)直接刑事追责+ 永久封号
  4. VAT 号无效 / 过期 = listing 全下架:野代办给的 VAT 号有时是”无效号”(税务局系统里查不到)或”过期号”,亚马逊每季度核验一次,查到无效 = 整个账号 24 小时内下架

反过来说,注册 VAT 的成本是 1000-2000 欧(代办费)+ 100-500 欧/次申报费。这是新手的”入门税”——绕不过去,但也别被坑。

2 种注册方式(代办 vs 自己)

方式 1:找当地税务代理代办(99% 新手的选择)

流程:

  1. 你找一家欧洲当地的税务代理(中国跨境圈常见的:Taxdoor、EasyTax、AVASK、Avalara、Vertex、BDO、Deloitte、KPMG 等)
  2. 提交材料(下面”5 步流程”详述)
  3. 代办帮你准备表格 + 提交到当地税务局
  4. 税务局审核 → 颁发 VAT 号
  5. 代办把 VAT 号给你 → 你填到亚马逊后台

费用:

  • 注册费:500-1500 欧 / 国家(一次性)
  • 申报费:100-500 欧 / 次(每月 / 每季度)
  • 年费汇总:5 国全开约 5000-15000 欧 / 年(看代理规模和你的销售额)

优点:

  • 省心——代办搞定所有文书 + 和税务局沟通
  • 语言无障碍——德国表格是德语,法国是法语,意大利是意大利语,代办都懂
  • 错误率低——专业代理 10 年经验,新手自己填错 = 拖延 2-4 周
  • 一次包过率 90%+——选择靠谱代理的话

缺点:

  • 贵——一年 1 万欧不是小数
  • 要挑代理——野代办(VAT 号无效)会坑死你,见下面”常见坑”

方式 2:自己远程注册(少数国家允许)

适用国家:英国(脱欧后仍可远程)、波兰(部分情况)、捷克(部分情况)。

流程:

  1. 自己上该国税务局网站(英国 gov.uk / 波兰 e-Urząd Skarbowy)
  2. 注册账户 + 填写申请表格(英文为主)
  3. 上传公司文件(营业执照、章程、法人身份证)
  4. 等 2-4 周审核,部分国家会视频面谈验证身份
  5. 拿到 VAT 号

费用:

  • 注册费:0(政府免费)
  • 申报费:0(自己填)或 50-200 欧(用免费软件如 TaxJar、Avalara)

优点:

  • 便宜——不付代理费
  • 流程可控——自己掌握时间

缺点:

  • 语言 + 法规门槛——德国、法国、意大利税务局不接受中国公司”无当地代表”远程注册(必须当地有财务代表 / Fiscal Representative)
  • 出错率高——表格填错、漏附件、错过申报 = 罚款
  • 部分国家不开放——德 / 法 / 意 / 西不接受中国公司直接远程申请,必须有 Fiscal Representative(当地财务代表,通常是代理)

结论:新手无脑找代办。自己远程注册只适合英国或有欧盟财务代表的情况。

5 步注册流程(从”啥也没”到”拿 VAT 号”)

步骤 1:确定要注册的国家(根据你的销售布局)

问自己 3 个问题:

  1. 你的 FBA 仓在哪些国家?(德国仓 → 德国 VAT 必办)
  2. 你在哪些 Amazon EU 站销售?(amazon.de / .fr / .it / .es / .nl)
  3. 你预估第一年的销售额?(超过某国 1 万欧 → 必注册)

新手标准配置:

  • 最少:德国 VAT(德国是欧洲最大市场,FBA 德国仓强制)
  • 标准:德 + 法 + 意(覆盖亚马逊欧洲 70% 销量)
  • 完整:德 + 法 + 意 + 西 + 荷(覆盖亚马逊欧洲 95% 销量)

步骤 2:准备注册材料(通常需要 4-6 项)

通用材料清单:

  1. 公司营业执照(中国大陆公司,中文 + 英文翻译件)
  2. 法人身份证 / 护照(正反面)
  3. 公司章程(中英文翻译件)
  4. 亚马逊店铺信息(店铺链接 + 邮箱 + 注册地址)
  5. 预计销售额 / 销售品类(有些国家要商业计划书)
  6. 授权书(Power of Attorney,授权代理替你申请)

特别注意:

  • 德国要求”Fiscal Representative”(当地财务代表)——通常是代办公司身份
  • 法国要求”Représentant Fiscal”——同上
  • 意大利要求”Rappresentante Fiscale”——同上
  • 西班牙、荷兰:可以”非欧盟卖家直接申请”,但实际还是要代办更省事
  • 所有文件做英文翻译(德国要德语,法国要法语,代办会处理)

步骤 3:提交申请 + 等待审核(2-6 周)

  • 提交后:税务局给你一个”申请号” / “case number”
  • 审核中:可能要求补充材料(2-4 周),或电话 / 视频面谈
  • 通过:发”VAT 证书”(PDF 文档,有你的 VAT 号 + 公司名 + 地址)
  • VAT 号格式:
    • 德国:DE123456789(DE + 9 位数字)
    • 法国:FRXX123456789(FR + 2 位校验码 + 9 位数字)
    • 意大利:IT12345678901(IT + 11 位数字)
    • 西班牙:ESX12345678(ES + 字母 + 7-8 位数字)
    • 荷兰:NL123456789B01(NL + 9 位数字 + B + 2 位)

步骤 4:把 VAT 号填到亚马逊后台

路径:Seller Central → Settings → Tax Information → VAT Information → Add VAT number

填入:

  • 国家(下拉选)
  • VAT 号(按上面格式)
  • 注册地址(欧代 / 代理地址或你公司欧盟地址)
  • 上传 VAT 证书 PDF

24-48 小时内亚马逊审核,通过后你的 listing 在该国站可正常销售 + 亚马逊不再代扣 VAT(你自己申报)。

步骤 5:开始申报(每季 / 每月一次)

申报流程(通常代办搞定):

  1. 代理在月初(15 日前)给你上月的销售报表
  2. 你核对(亚马逊后台 → Reports → Tax Reports 下载)
  3. 代理根据报表填申报表 + 提交到税务局
  4. 税务局核算应交 VAT
  5. 你打款到税务局账户(代理会告诉你金额 + 账户)
  6. 代理发你”申报回执”存档

申报频率速查:

国家默认频率升级到月报的条件
德国季报上年 VAT 应缴 > 7500 €
法国月报(默认)无,强制月报
意大利季报上年 VAT 应缴 > 50 万 €
西班牙月报无,强制月报
荷兰季报自行申请可改月报

OSS 模式(跨境小包 < 150 € 的简化方案)

OSS(One-Stop Shop,一站式申报):欧盟 2021-07-01 推出的简化方案,适合跨境小包(从中国直邮,单件 < 150 €)。

核心逻辑:

  • 你只需要在一个欧盟国家(比如荷兰)注册一个 OSS VAT 号
  • 然后全欧盟销售都通过这一个号申报——不用挨个国家注册
  • 税率按买家所在国计算(德国买家 → 19%,法国买家 → 20%)

适合:

  • FBM(自发货)+ 跨境小包 + 单件 < 150 € 的卖家
  • 不想挨个国家注册 + 申报的轻量卖家
  • Dropshipping / 无货源模式

不适合:

  • FBA 库存放在欧盟境内(如德国 FBA 仓)→ 必须单独注册德国 VAT,不能用 OSS 替代
  • 单件 > 150 € 的高价值商品 → 不能用 IOSS(下面)

IOSS(Import OSS):OSS 的子集,专门针对进口商品 < 150 €:

  • IOSS 号让海关快速清关(避免买家被收进口 VAT)
  • 同样只需要一个号,全欧盟通用
  • 适合纯跨境小包业务

实操建议:

  • FBA 卖家:老老实实德 + 法 + 意 VAT 必办,OSS 用不上
  • FBM 跨境小包为主:可以只注册一个 OSS 号,省去多国注册麻烦
  • 混合模式:FBA VAT + OSS 同时存在,代理会告诉你怎么申报

4 个常见坑(新手 100% 会踩)

坑 1:贪便宜找”野代办”,VAT 号无效 → listing 全下架

这是新手最常被坑的点——

野代办特征:

  • 价格明显低于市场(200-500 欧包注册,正常是 1000+)
  • 没有实际办公室,只有 QQ / 微信沟通
  • 承诺”3 天拿号”(正常 4-6 周,3 天是伪造号)
  • VAT 号查不到(给你一个”DE123456789”,你去欧盟税务局网站查不到)
  • 没有申报服务,只负责”注册”

被坑后果:

  • 亚马逊每季度核验一次 VAT 号有效性,查到无效 = 24 小时内 listing 全下架
  • 重新注册合法 VAT 号要 4-6 周,期间销售归零
  • 已经销售的款项可能被税务局追缴

怎么避:

  • 选有欧盟实体办公室的代理(查办公地址,LinkedIn 看团队)
  • 行业知名代理:Taxdoor、EasyTax、AVASK、Avalara、Vertex(国际四大)、Sovos
  • 不要贪便宜——VAT 注册费 800-1500 欧是合理范围,低于 500 欧的警惕
  • VAT 号拿到手立刻去欧盟 VIES 系统验证:https://ec.europa.eu/taxation_customs/vies/

坑 2:不报税 / 漏报税,等到税务局来敲门才慌

VAT 是”自行申报”,不是”等通知”。你不主动报,税务局会:

  • 第 1 个月没报 → 警告信
  • 第 3 个月没报 → 罚款 + 加收滞纳金(每月 1-2% 利息)
  • 第 6 个月没报 → 严重罚款(销售额的 5-30%)+ 刑事追责

新手 100% 漏报的场景:

  • 销售了 0 欧 / 几欧也懒得申报 → 错! 即便 0 销售也要 0 申报
  • 换了代理,新旧代理之间”申报真空期” → 要书面交接
  • 注销账号前没做”最终申报”(Final Return) → 税务局会追缴

怎么避:

  • 设月度提醒(Google Calendar / Notion),每月 10 日检查”上月是否申报”
  • 0 销售也要申报(申报表填 0,不是不报)
  • 换代理前:让旧代理做完最终申报,新代理接手,中间无真空期
  • 注销账号前:做”Final Return”(最终申报),申报完才会”清税”

坑 3:多国注册不全,亚马逊代扣后,自己的 VAT 号抵不了

新手常犯:只注册了德国 VAT,没有法国 / 意大利,但listing 同时挂在 amazon.de / .fr / .it 三个站点。结果:

  • 德国销售:你自己申报(OK)
  • 法国 / 意大利销售:亚马逊直接代扣 20-22% VAT(你没 VAT 号,亚马逊代扣不抵减)
  • 多付 3-5% 的税,白白给亚马逊当”过路财神”

怎么避:

  • Amazon 欧洲站多店 + 多国销售 → VAT 多国注册(德 + 法 + 意 / 西 / 荷)
  • 算账:亚马逊代扣 VAT 比例 vs 你自己注册 + 申报的成本,通常自己注册更划算
  • 暂时只做一国:关掉其他国家的 listing(在 Seller Central → Inventory → 关 listing),不要让它”自然销售”

坑 4:注册后乱开”零税率发票”,被认定为逃税

VAT 跨境 B2B 销售有个”零税率”机制:

  • 你卖 1000 欧商品给德国企业买家 → VAT 0%(对方走反向征税,自己申报)
  • 你要在发票上明确写 “Reverse Charge” + 对方 VAT 号

新手 100% 踩坑:

  • 什么客户都开 0% 发票 → 错!(个人买家不能开 0%,企业买家要有 VAT 号)
  • 对方给了”假 VAT 号”(野代办那种),你也照开 0% → 错!(你有责任验证对方 VAT 号有效性)
  • 不验证对方 VAT 号就开 0% 发票 → 税务局查账时,你这部分 VAT 全部要补交 + 罚款

怎么避:

  • 开 0% 发票前必做 2 件事:
    1. 验证对方 VAT 号:用 VIES 系统(https://ec.europa.eu/taxation_customs/vies/)
    2. 拿到对方”Intra-Community VAT Number”(欧盟内部交易 VAT 号)
  • 个人买家一律开含税发票(标准税率 19-22%)
  • 每月导出”零税率发票清单”(VAT MOSS report),留底 10 年备查

何时注册 / 何时不注册

必须注册 VAT 的情况

  • 在欧盟任何国家有库存(FBA 仓或自建海外仓)
  • 跨境销售到欧盟个人买家(B2C)
  • 跨境 B2B 销售(从中国直发到欧盟企业)
  • 使用亚马逊 pan-EU(泛欧计划)——亚马逊自动调拨库存,触发所有相关国家 VAT 注册
  • 销售额预期 > 1 万欧 / 年 / 国家(欧盟”小微企业”门槛)

可以不注册 VAT 的情况(谨慎判断)

  • 纯样品 / 测试品(不销售)——但海关有时仍要 VAT 号清关
  • 跨境小包 < 150 € 用 IOSS(一个号全欧通用)——但只对 FBM 直邮有效
  • B2B 走反向征税 + 对方有欧盟 VAT 号——理论上你可以不注册,但实际海关和亚马逊仍要求你注册

判断不了? 直接问税务代理,或者开 Amazon case 问。

FAQ(常见问题)

Q1:我有 1 个中国公司,能在 5 个国家同时注册 VAT 吗? A:能,但每个国家都要单独申请 + 单独审核。你的中国公司是同一个主体,但VAT 号按国家分别颁发:

  • 德国 VAT 号 = DE + 9 位
  • 法国 VAT 号 = FR + 11 位
  • 意大利 VAT 号 = IT + 11 位
  • 5 个号同时存在,各自独立申报

Q2:亚马逊”Marketplace Facilitator”代扣后,我还需要自己注册 VAT 吗? A:要,即使被代扣。原因:

  • 被代扣不等于”免税”——亚马逊代扣的 VAT 是按”全额售价”算的(没抵减你的进货 VAT)
  • 你自己注册后,可以申报时抵减进货 VAT,实际税负 = 销售额 VAT - 进货 VAT(差额)
  • 长期被代扣 = 你的利润被多吃 1-3%
  • 结论:被代扣也要注册,否则你白白多交税

Q3:VAT 注册后,如果不销售了,可以注销吗? A:可以,但要”清税”:

  • 做”Final Return”(最终申报):把所有应交 VAT 上交
  • 申报完后,通知税务局”申请注销”
  • 注销后 5-10 年内,税务局仍有权查你的申报记录(资料保留 10 年是欧盟硬性要求)
  • 注销 ≠ 销户:VAT 号注销后,亚马逊后台这个号变”已注销”,但你的账号仍能开新 VAT 号

Q4:申报 VAT 时,我的”销售收入”按哪个金额算? A:按亚马逊后台”含税销售额”算(Gross Sales,包含 VAT 的金额)。

  • 例:你在 amazon.de 卖 100 件,单价 30 €(含 19% VAT)
  • 总销售额(含税)= 3000 €
  • 应交 VAT = 3000 × 19/119 = 478.99 €(用 19/119 不是 19%,因为售价里已经含 VAT 了)
  • 别用 3000 × 19% = 570 € 这种算法(重复计税)

Q5:VAT 申报可以用 Excel 自己算吗?要不要买专业软件? A:可以,但要严格:

  • 新手前 3 个月:用 Excel 模板,自己算 + 手动提交(代办会教你)
  • 3 个月后:用专业软件如 TaxJar、Avalara、Vertex、Sovos(订阅费 50-200 欧 / 月)
  • 亚马逊后台 → Reports → Tax Reports 自动生成 CSV → 导入软件 → 自动算 → 提交
  • Excel 自己算的坑:四舍五入错误、币种换算错误、负数(VAT 抵减)处理错误

Q6:我可以用朋友的 VAT 号吗? A:绝对不行。VAT 号和公司是绑定的,别人的号 = 别人的税。用别人的号 = 双重违法(你的偷税 + 朋友的可能被牵连)。自己注册一个,别省这个钱。

Q7:我刚注册 VAT 号,还没销售,要不要 0 申报? A:要 0 申报。即使 0 销售,也要按时报:

  • 申报表填”0 € 销售 / 0 € VAT 应缴”
  • 不报 = 漏报 = 罚款(同上”漏报”章节)
  • 0 申报的意义:证明你这段时间”没偷税”

行动清单(VAT 注册到开卖,4 周搞定)

按这个清单走,4-6 周拿号,30 天内开卖:

  • 第一步(第 1 周):确定要注册的国家(德国是底线,法 / 意 / 西 / 荷按销售布局)
  • 第二步(第 1 周):挑 1 家靠谱税务代理(查办公地址、看 LinkedIn 团队、问同行推荐)
  • 第三步(第 2 周):提交材料(营业执照、身份证、店铺信息、授权书)
  • 第四步(第 2-6 周):等待审核,代理跟进,补充材料(如有)
  • 第五步(拿到号):VAT 号去 VIES 系统验证 + 填到亚马逊后台(Settings → Tax Information)
  • 第六步(开卖前):把 VAT 号同步到所有 listing(亚马逊会自动抓取)
  • 第七步(每月 / 每季):核对销售报表 + 申报(代办) + 打款到税务局
  • 第八步(每年):1 月做”年度申报”(Annual Return,德国要,法 / 意可免)
