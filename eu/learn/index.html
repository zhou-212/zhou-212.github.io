学习路径

把站内 23 张卡片 + 10 个对比 + 10 个入门课 + 3 个清单, 按"新手最容易踩的 5 个坑"重新串成 5 条路径。 每条 5-6 步,总计 50-90 分钟 读完, 进度自动存在你浏览器里,下次回来接着走。

先做影响自测 下载个人合规清单 PDF
🌱 6 步 · 约 90 分钟

新手起步(0-90 天)

从注册账号到首单出单,90 天走完基本合规

适合:第一次做亚马逊欧洲站的卖家

完成后:账号能开张 + 基础合规齐 + 首批货上架
💰 6 步 · 约 60 分钟

税务与申报

VAT / IOSS / OSS 三条路,1 张表说清怎么选

适合:FBA 卖家 + 跨境直邮卖家

完成后:VAT/IOSS 选型 + 申报流程清晰 + 不会因税务问题被海关扣货
🛡️ 6 步 · 约 80 分钟

产品安全(GPSR / CE / CRA)

从 CE 标志到 CRA 网络安全,5 步把产品合规做齐

适合:非食品消费品卖家 + 联网产品卖家

完成后:CE 标志 + 欧代 + DoC + CRA SBOM 全齐
♻️ 6 步 · 约 70 分钟

包装与环保

从 LUCID 到电池护照,德国 LUCID + 法国 EPR + PPWR 一次性说清

适合:所有带包装的卖家 + 含电池商品卖家

完成后:LUCID / Citeo 注册齐 + 包装检测齐 + 电池护照准备齐
⚙️ 5 步 · 约 50 分钟

运营合规(广告 / 评论 / 警告信)

5 类警告信应对 + 广告合规词 + 评论合规

适合:已开张要稳定出单的卖家

完成后:警告信能应对 + 广告投放不踩雷 + 评价合规积累
🚨 5 步 · 约 60 分钟

危机公关(收到警告信 / 限流 / 扣货怎么救)

5 步救店:识别 → 48h 应对 → 申诉模板 → 复盘 → 预防

适合:所有遇到 / 可能遇到合规危机的亚马逊欧洲站卖家

完成后:能冷静应对 5 大危机 + 申诉邮件模板 + 48 小时应对清单 + 复盘模板
🎯 5 步 · 约 60 分钟

选品合规调研

亚马逊欧洲站选品前 60 分钟的合规快速排查

适合:新卖家 / 选品新手 / 转类目老卖家

完成后:走完 60 分钟,你会得到:1 张"产品合规检查表"+ 1 个"能不能卖"的明确判断(卖 / 缓卖 / 不卖)+ 1 个合规总成本预算。这是新品上市前的最后一道门,带去采购/选品/小组成员会直接能用。
提示:学习路径进度存在 localStorage 里, 不会上传服务器,清浏览器数据会重置。如果你想要"长期可恢复的进度" + 邮件提醒, 留 email 订阅 每周合规更新。
