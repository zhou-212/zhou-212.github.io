PDF 合并 — 多个 PDF 拼成一个,浏览器本地完成

拖几个 PDF 进来,排好顺序,点一下就合成一个新文件。所有东西都在你浏览器里跑, 文件不离开你的电脑,也不需要注册。

  1. 1把 PDF 拖到下面那个大框里,或者点框选文件
  2. 2按住左边的 ⠿ 拖动,调整文件前后顺序
  3. 3点「合并 PDF」,等几秒钟,下载合好的新文件

    为什么用这个工具?

    你可能遇到过:把发票、合同、检测报告拼成一份给亚马逊或代理,或者把多份 PDF 上传到一个工单。 不少在线 PDF 工具会先把文件传到他们的服务器——你的商业文件、报价单、客户信息,别人都看得见。

    这个工具完全在浏览器里跑,用的是 pdf-lib 这个开源库, 点完「合并」,文件不经过任何第三方。可以随时关掉网络,功能照样能用。 上传多少文件、文件多大,都不限制。
