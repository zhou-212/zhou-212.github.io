隐私说明

最后更新:2026-07-13

欧规雷达(以下简称"本站")尊重你的隐私。本站使用 GitHub Pages 静态托管,第一阶段不部署后端服务,也不内置第三方追踪代码。访问本站时,你的数据由 GitHub Pages 和浏览器 / DNS 提供方按其各自隐私政策处理。

1. 我们不收集的信息

  • 不收集姓名、电话、地址等个人识别信息
  • 不收集你的店铺、订单、SKU 等商业数据
  • 不收集支付 / 银行卡 / 凭证信息(本站不收费)
  • 不在站内保存你的勾选 / 自测结果到服务器(清单勾选仅保存在你本地浏览器)

2. 我们可能收集的信息

(1) 浏览器本地存储(localStorage)

行动清单的勾选状态会保存在你的浏览器本地(localStorage),用于下次访问恢复进度。删除浏览器数据会清除这些记录。清除不影响其它浏览器或设备。

(2) 邮件订阅

当前状态:订阅功能筹备中,当前不收集、不保存、不发送任何邮箱数据。 站内的订阅表单按钮在配置完成前为禁用状态;若你尝试提交,浏览器会阻止,且不会发出任何网络请求。

启用条件:运营在 public/runtime-config.js 填入真实的 API_BASE_URL 后,表单会变为可用,提交的邮箱数据会通过 POST {API_BASE_URL}/api/v1/subscriptions 发送到自建后端。在此之前,本站不会调用任何第三方邮件服务(Tally / Brevo / ConvertKit 等)。

启用后,数据将由你的自建后端按你自己的隐私政策处理;本站只作为前端,不对后端数据保存或传输承担责任。

(3) 访问统计(可选)

本站不内置任何分析代码。运营者可能通过 GitHub Pages 的基础访问日志查看总访问量(不包含个人识别信息)。

如未来启用第三方统计(例如 Plausible / Umami 自托管 / Cloudflare Analytics),会在此页面更新并明确告知。

3. Cookie 使用

本站是静态站点,本身不设置任何 Cookie。如果你启用第三方分析或邮件服务,可能引入第三方 Cookie;届时会明确告知。

4. 第三方服务

本站可能链接到:

  • Amazon Seller Central、欧盟委员会 / EUR-Lex 等官方源(阅读时按对方隐私政策处理)
  • 自建后端 API(订阅功能启用后;位于 {API_BASE_URL} 替换后的真实地址)
  • 第三方分析服务(如启用)

我们不对第三方服务的隐私实践承担责任。

5. 儿童隐私

本站面向商业用户,不针对 14 岁以下儿童。我们不故意收集儿童信息。

6. 数据保留

本站不维护用户数据库;订阅启用后数据由你的自建后端管理,你可在后端随时查看 / 删除。清单勾选数据仅在你的浏览器本地。

7. 你的权利

你可以随时:

  • 清除浏览器 Cookie / localStorage(本站无 Cookie,只有 localStorage 中的清单勾选)
  • 在自建后端管理界面查看 / 删除 / 导出 / 退订你的订阅数据(启用后)
  • 停止使用本站,无需告知

8. 跨境传输

本站通过 GitHub Pages 托管,服务器可能位于美国 / 欧盟以外。订阅启用后,邮箱数据的传输由自建后端所在地区处理。

9. 条款更新

我们可能在必要时更新本隐私说明,更新会在本站"最后更新"字段标注。

10. 联系

对隐私有疑问,可联系:YOUR_CONTACT_EMAIL@example.com(占位,部署前替换为你的真实邮箱)。
