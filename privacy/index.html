隐私说明

最后更新:2026-07-15

欧规雷达(以下简称"本站")尊重你的隐私。本站使用 GitHub Pages 静态托管,第一阶段不部署后端服务,也不内置第三方追踪代码。访问本站时,你的数据由 GitHub Pages 和浏览器 / DNS 提供方按其各自隐私政策处理。

1. 我们不收集的信息

  • 不收集姓名、电话、地址等个人识别信息
  • 不收集你的店铺、订单、SKU 等商业数据
  • 不收集支付 / 银行卡 / 凭证信息(本站不收费)
  • 不在站内保存你的勾选 / 自测结果到服务器(清单勾选仅保存在你本地浏览器)

2. 我们可能收集的信息

(1) 浏览器本地存储(localStorage)

行动清单的勾选状态会保存在你的浏览器本地(localStorage),用于下次访问恢复进度。删除浏览器数据会清除这些记录。清除不影响其它浏览器或设备。

(2) 邮件订阅

当前状态:订阅服务正在接入,接入完成前不收集、不保存、不发送任何邮箱数据。 站内的订阅表单在 __MAILCHIMP_ACTION__ / __BREVO_ACTION__ 占位替换前,点击提交会直接走兜底提示(让用户直接发邮件到 i@weiread.com),不会向任何第三方邮件服务发送数据。

接入完成后,数据将由 Mailchimp / Brevo(已签 DPA + SCC)按其各自隐私政策处理;本站仅作为前端,不对邮件服务的数据保存或传输承担责任。详细数据处理规则见下方第 11 节"📧 邮件订阅数据处理"。

(3) 访问统计(可选)

本站不内置任何分析代码。运营者可能通过 GitHub Pages 的基础访问日志查看总访问量(不包含个人识别信息)。

如未来启用第三方统计(例如 Plausible / Umami 自托管 / Cloudflare Analytics),会在此页面更新并明确告知。

3. Cookie 使用

本站是静态站点,本身不设置任何 Cookie。如果你启用第三方分析或邮件服务,可能引入第三方 Cookie;届时会明确告知。

4. 第三方服务

本站可能链接到:

  • Amazon Seller Central、欧盟委员会 / EUR-Lex 等官方源(阅读时按对方隐私政策处理)
  • 自建后端 API(订阅功能启用后;位于 {API_BASE_URL} 替换后的真实地址)
  • 第三方分析服务(如启用)

我们不对第三方服务的隐私实践承担责任。

5. 儿童隐私

本站面向商业用户,不针对 14 岁以下儿童。我们不故意收集儿童信息。

6. 数据保留

本站不维护用户数据库;订阅启用后数据由你的自建后端管理,你可在后端随时查看 / 删除。清单勾选数据仅在你的浏览器本地。

7. 你的权利

你可以随时:

  • 清除浏览器 Cookie / localStorage(本站无 Cookie,只有 localStorage 中的清单勾选)
  • 在自建后端管理界面查看 / 删除 / 导出 / 退订你的订阅数据(启用后)
  • 停止使用本站,无需告知

8. 跨境传输

本站通过 GitHub Pages 托管,服务器可能位于美国 / 欧盟以外。订阅启用后,邮箱数据的传输由自建后端所在地区处理。

9. 条款更新

我们可能在必要时更新本隐私说明,更新会在本站"最后更新"字段标注。

10. 联系

对隐私有疑问,可联系:i@weiread.com。我们会在 30 天内回复。

11. 📧 邮件订阅数据处理

如果你订阅了我们的每周新规摘要邮件,以下说明我们如何收集、使用、存储和保护你的数据。本节遵循 GDPR 第 13、15、17 条信息告知义务。

11.1 我们使用什么服务

邮件订阅服务由 Mailchimp(默认)或 Brevo(备份)提供,二者均为 GDPR 合规的邮件营销服务商,已签署数据处理协议(DPA)并采用欧盟标准合同条款(SCC)。

占位:实际接入服务以本页顶部"最后更新"日期为准;Mailchimp / Brevo 接入前不收集任何邮箱数据。

11.2 我们收集什么

  • 邮箱地址(必填,用于发送周报)
  • 提交时的 IP 地址和时间戳(由邮件服务商记录,用于反垃圾和异常检测,不用于画像)
  • GDPR opt-in 勾选状态(用于证明你已明确同意接收邮件,符合 GDPR Art. 7)

11.3 怎么用

你的邮箱仅用于:每周三上午(北京时间)发送一条欧盟新规摘要(欧规雷达 周报)。

不会用于:

  • 发送广告、第三方促销或联盟营销邮件
  • 出售或出租给任何第三方
  • 与你的网站访问行为、你的产品或你的店铺数据交叉关联

11.4 怎么退订

每封邮件底部都包含一个 unsubscribe 链接(由 Mailchimp / Brevo 自动生成),点击立即生效,不需要登录账号,不需要理由。

退订后 30 天内,你的邮箱会从主发件列表中删除;但保留在"已退订"档案中一段时间以防误发(GDPR Art. 17 允许的合法处理例外)。

11.5 你的权利(GDPR Art. 13 / 15 / 17)

你可以随时行使以下权利:

  • 查阅(Art. 15):写信给 i@weiread.com,我们 30 天内导出你订阅时的全部记录(邮箱 / IP / 时间 / 同意状态)
  • 更改为同一个邮箱(Art. 16):拼错的邮箱、想换收件地址,邮件告知即可
  • 删除(Art. 17):同上邮件告知,我们 30 天内从主列表清除;已退订的也会在 30 天内从档案中彻底删除
  • 撤回同意(Art. 7(3)):等同于退订,一键完成
  • 投诉(GDPR Art. 77):如认为处理违反 GDPR,可向所在国数据保护监管机构投诉

11.6 政策生效日期

本节内容自 2026-07-15 起生效;接入真实邮件服务后,我们会在此页面更新"最后更新"日期。
