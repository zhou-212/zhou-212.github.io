亚马逊欧洲站新卖家基础资料检查清单

第一次在 Amazon EU 开店或扩展到新国家前必须准备的核心资料 —— 法人、税务、知识产权、欧盟责任人、合规文件等。

适用对象
新卖家、扩展到新 EU 国家的卖家、多平台运营者
最后核对
2026年1月15日
标签
新卖家、开店、VAT、EPR、责任人
进度: 0%

适用范围

  • 新注册 Amazon EU 任一国家站点的卖家
  • 已经在某 EU 国家开张,扩展到第二个 EU 国家的卖家
  • 多平台运营 同步需求

最后核对日期

2026-01-15 —— 详见本清单每条来源链接;请在每次开店前再次核对最新规则。

来源提示与免责声明

本清单来自 Amazon Seller Central 公开页、欧盟委员会、成员国官方机构。清单本身不构成法律或税务意见;具体国家、类目、税务、平台要求以官方原文为准。

1. 法人 / 公司主体

  • 注册公司主体(可接受中国 / 香港 / 美国公司)
  • 公司银行账户(支持 EUR 收款的账户,例如 WorldFirst、Payoneer、PingPong)
  • 公司章程 / 注册证明(可被平台和银行审核)

2. 欧盟 VAT(增值税)

  • 申请欧盟 VAT 号(在仓储国和/或销售国)
  • 在 Amazon 后台绑定 VAT 号
  • 了解 OSS / IOSS 制度(取决于你的发货模式)
  • 评估 VAT 申报频率(月度 / 季度 / 年度)
  • 委托 VAT 申报代理或自建内部流程

3. 欧盟责任人(EU Responsible Person)

  • 选择 / 任命 EU 责任人(关联公司 / Authorised Representative 服务)
  • 签订书面责任协议
  • 将责任人信息(姓名、地址、电子邮箱)录入 Amazon 后台
  • 在产品 / 包装上印刷责任人信息(可识别)

4. 包装与 EPR(生产者延伸责任)

  • 德国 LUCID 注册 + 回收系统参与
  • 法国 Citeo(包装) + Ecologic(电气) + 其它适用类目 EPR
  • 西班牙 ECOEMBES / SIGRE 等
  • 意大利 CONAI 等
  • 波兰 、荷兰 等其它 EU 国家类似
  • 包装 Triman / 回收标识印刷

5. 知识产权(IP)

  • 品牌注册(Amazon Brand Registry 必备:欧盟商标或本地商标)
  • 专利、版权、设计权自我清查
  • 代理授权(若是代理品牌)

6. 产品合规文件(每个 ASIN / 类目)

  • CE 标志 + EU Declaration of Conformity(适用产品)
  • 测试报告(EN 71、EN 62115、EMC、LVD 等,按类目)
  • 安全数据表 SDS(化学品 / 化妆品)
  • 食品接触材料声明(FCM)
  • 化妆品产品信息文件(PIF)
  • 用户说明书 + 多语言版本(德/法/意/西/英)

7. 平台设置

  • 创建 Seller Central 账户
  • 完成身份验证(KYC)
  • 配置物流(FBA / FBM / 多渠道)
  • 设定语言、货币、客服联系方式
  • 客服联系方式需含当地语言(德语 / 法语 / 当地官方语言)
  • 退换货政策(各站点不同)

8. 财务与运营

  • 财务对账流程(平台收款 - VAT - 平台费 - 物流费 - 退款)
  • 库存管理(尤其多国 FBA 调拨)
  • 客服响应时效(Amazon EU 通常要求 24-48h 邮件响应)
  • 多账号合规(Amazon 多账号政策极严,提前评估)

9. 持续监控

  • 订阅本雷达每周汇总
  • 监控 Amazon Seller Central 公告
  • 监控销售国市场监督机构(MSAs)新闻
